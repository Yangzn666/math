二、核心概念

1. 物理应用（仅数学一、数学二）

相关变化率问题

基本概念：

相关变化率：两个或多个变量之间的变化率有关系

解题思路：

建立相关变量方程 → 求导找出相关变化率 → 通过已知变化率求未知变化率

典型例题：

位移对时间的变化率（速度）
速度对时间的变化率（加速度）

2. 经济应用（仅数学三）

(1) 边际函数与边际分析

边际成本：\(MC(Q) = C'(Q)\)
边际收益：\(MR(Q) = R'(Q)\)
边际利润：\(ML(Q) = L'(Q) = MR(Q) - MC(Q)\)

边际分析的经济意义：

边际利润大于0：增加产量可以增加利润
边际利润小于0：减少产量可以增加利润
边际利润等于0：利润达到最大值

(2) 弹性函数与弹性分析

需求弹性：\(\eta = - \frac{p}{Q} \cdot \frac{dQ}{dp}\)

弹性分析：

\(\eta > 1\)：需求富有弹性，价格下降会增加总收益
\(\eta = 1\)：需求单位弹性，价格变化不影响总收益
\(\eta < 1\)：需求缺乏弹性，价格下降会减少总收益

三、常见题型及解题思路

1. 经济应用问题（边际分析）

解题思路：

1. 根据题目条件，写出成本函数、收益函数、利润函数
2. 求出边际利润函数
3. 代入条件求解

例：设生产某商品的固定成本为60000元，可变成本为20元/件，价格函数为\(p = 60 - Q/1000\)（\(p\)是单价，单位：元；\(Q\)是销量，单位：件）。已知产销平衡，求：

(1) 该商品的边际利润函数；
(2) 当\(p = 50\)元时的边际利润，并解释其经济意义；
(3) 使得利润最大的单价\(p\)。

解题步骤：

成本函数：\(C(Q) = 60000 + 20Q\)
收益函数：\(R(Q) = pQ = (60 - Q/1000)Q = 60Q - Q^2/1000\)
利润函数：\(L(Q) = R(Q) - C(Q) = -Q^2/1000 + 40Q - 60000\)
边际利润函数：\(L'(Q) = -Q/500 + 40\)
当\(p = 50\)元时：
\(50 = 60 - Q/1000 \Rightarrow Q = 10000\)
\(L'(10000) = -10000/500 + 40 = 20\)元
经济意义：销售第10001件商品所得利润为20元
利润最大时：
令\(L'(Q) = 0 \Rightarrow -Q/500 + 40 = 0 \Rightarrow Q = 20000\)
代入价格函数：\(p = 60 - 20000/1000 = 40\)元

2. 弹性分析问题

解题思路：

1. 写出需求函数\(Q = Q(p)\)
2. 计算需求弹性\(\eta = - \frac{p}{Q} \cdot \frac{dQ}{dp}\)
3. 根据弹性大小判断价格变动对总收益的影响

例：设某商品需求量\(Q\)是价格\(p\)的单调减少函数：\(Q = Q(p)\)，其中需求弹性\(\eta = \frac{2p^2}{22 - p^2} > 0\)。设\(R = R(p)\)为总收益函数，证明：\(\frac{dR}{dp} = Q(1 - \eta)\)

解题步骤：

总收益：\(R = pQ\)
求导：\(\frac{dR}{dp} = Q + p\frac{dQ}{dp}\)
由需求弹性：\(\eta = - \frac{p}{Q}\frac{dQ}{dp}\)，可得\(p\frac{dQ}{dp} = -\eta Q\)
代入得：\(\frac{dR}{dp} = Q - \eta Q = Q(1 - \eta)\)

四、易错点与注意事项

1. 边际函数的定义

边际成本是成本函数对产量的导数，不是平均成本
边际收益是收益函数对产量的导数，不是平均收益

2. 经济意义的理解

边际利润大于0，增加产量可增加利润
边际利润小于0，减少产量可增加利润
边际利润等于0，利润达到最大值

3. 需求弹性与总收益的关系

需求富有弹性（\(\eta > 1\)）：价格下降 → 总收益增加
需求缺乏弹性（\(\eta < 1\)）：价格下降 → 总收益减少
需求单位弹性（\(\eta = 1\)）：价格变动 → 总收益不变

4. 相关变化率问题

建立相关变量方程是关键
求导时要正确应用链式法则
需要明确已知变化率和未知变化率

5. 价格函数与需求函数的关系

价格函数\(p = p(Q)\)：价格是销量的函数
需求函数\(Q = Q(p)\)：销量是价格的函数
两者互为反函数

五、重要公式汇总

1. 边际函数：
    \(MC(Q) = C'(Q)\)
    \(MR(Q) = R'(Q)\)
    \(ML(Q) = L'(Q) = MR(Q) - MC(Q)\)
2. 需求弹性：
    \(\eta = - \frac{p}{Q} \cdot \frac{dQ}{dp}\)
    总收益\(R = pQ\)
    \(\frac{dR}{dp} = Q(1 - \eta)\)
3. 利润最大化条件：
    令\(ML(Q) = 0\)，即\(MR(Q) = MC(Q)\)
4. 相关变化率问题：
    建立相关变量方程
    求导找出相关变化率
    通过已知变化率求未知变化率

六、典型例题解析

例1：边际利润问题

题目：设生产某商品的固定成本为60000元，可变成本为20元/件，价格函数为\(p = 60 - Q/1000\)（\(p\)是单价，单位：元；\(Q\)是销量，单位：件）。已知产销平衡，求：

(1) 该商品的边际利润函数；
(2) 当\(p = 50\)元时的边际利润，并解释其经济意义；
(3) 使得利润最大的单价\(p\)。

解题思路：

成本函数：\(C(Q) = 60000 + 20Q\)
收益函数：\(R(Q) = pQ = (60 - Q/1000)Q = 60Q - Q^2/1000\)
利润函数：\(L(Q) = R(Q) - C(Q) = -Q^2/1000 + 40Q - 60000\)
边际利润函数：\(L'(Q) = -Q/500 + 40\)
当\(p = 50\)时：\(50 = 60 - Q/1000 \Rightarrow Q = 10000\)
\(L'(10000) = -10000/500 + 40 = 20\)元
经济意义：销售第10001件商品所得利润为20元
利润最大时：\(L'(Q) = 0 \Rightarrow Q = 20000\)
\(p = 60 - 20000/1000 = 40\)元

例2：需求弹性问题

题目：设某商品需求量\(Q\)是价格\(p\)的单调减少函数：\(Q = Q(p)\)，其中需求弹性\(\eta = \frac{2p^2}{22 - p^2} > 0\)。设\(R = R(p)\)为总收益函数，证明：\(\frac{dR}{dp} = Q(1 - \eta)\)

解题思路：

总收益：\(R = pQ\)
求导：\(\frac{dR}{dp} = Q + p\frac{dQ}{dp}\)
由需求弹性定义：\(\eta = - \frac{p}{Q}\frac{dQ}{dp} \Rightarrow p\frac{dQ}{dp} = -\eta Q\)
代入得：\(\frac{dR}{dp} = Q - \eta Q = Q(1 - \eta)\)
