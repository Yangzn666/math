二、核心概念

1. 重要结论

重要极限：\(\lim_{t \to 0^+} t^a \ln t = 0 \quad (a > 0)\)

证明：设\(I = \int_0^1 t^a \ln t \, dt = \lim_{\varepsilon \to 0^+} \int_\varepsilon^1 t^a \ln t \, dt\)

用分部积分法：设\(u = \ln t, dv = t^a dt\)

\(du = \frac{1}{t}dt, \quad v = \frac{t^{a+1}}{a+1}\)
\(I = \left[\frac{t^{a+1}}{a+1} \ln t\right]_\varepsilon^1 - \int_\varepsilon^1 \frac{t^{a+1}}{a+1} \cdot \frac{1}{t} dt\)
\(= 0 - \lim_{\varepsilon \to 0^+} \left[\frac{t^{a+1}}{a+1} \ln t\right]_\varepsilon^1 - \int_\varepsilon^1 \frac{t^a}{a+1} dt\)
\(= 0 - 0 - \left[\frac{t^{a+1}}{(a+1)^2}\right]_\varepsilon^1\)
\(= 0 - 0 - \left[\frac{1}{(a+1)^2} - 0\right] = -\frac{1}{(a+1)^2}\)

由夹逼准则得：\(\lim_{n \to \infty} \int_0^1 x^n f(x)dx = 0\)，其中\(f(x)\)在\([0,1]\)上连续

2. 积分中值定理

若函数\(f(x)\)在区间\([a,b]\)上连续，则至少存在一点\(\xi \in [a,b]\)，使\(\int_a^b f(x)dx = f(\xi)(b-a)\)

3. 夹逼准则

若存在三个函数\(f(x), g(x), h(x)\)，使得\(f(x) \leq g(x) \leq h(x)\)，且\(\lim_{x \to x_0} f(x) = \lim_{x \to x_0} h(x) = A\)，则\(\lim_{x \to x_0} g(x) = A\)

三、常见题型及解题思路

1. 证明积分等式

解题思路：

1. 分析被积函数的性质
2. 选择适当的积分方法（如分部积分、换元法）
3. 利用已知条件进行推导
4. 通过恒等变形得出所需等式

例11.6：设\(f(x)\)的二阶导数\(f''(x)\)在\([0,1]\)上连续，且\(f(0) = f(1) = 0\)，证明：

(1) \(\int_0^1 f(x)dx = -\int_0^1 x(x-1)f''(x)dx\)
(2) \(\int_0^1 f(x)dx \leq \frac{1}{12}\max f''(x)\)

解题步骤：

证明(1)：
用两次分部积分法：
\(\int_0^1 f(x)dx = \int_0^1 1 \cdot f(x)dx\)
\(= [x f(x)]_0^1 - \int_0^1 x f'(x)dx\)
\(= 0 - [x f'(x)]_0^1 + \int_0^1 f'(x)dx\)
\(= 0 - [1 \cdot f'(1) - 0] + [f(x)]_0^1\)
\(= -f'(1) + 0 = -f'(1)\)
另一方面：
\(\int_0^1 x(x-1)f''(x)dx = [x(x-1)f'(x)]_0^1 - \int_0^1 (2x-1)f'(x)dx\)
\(= 0 - [(x^2 - x)f'(x)]_0^1 + \int_0^1 (2x-1)f(x)dx\)
\(= -[0 - 0] + [x^2 - x]f(x) _0^1 - \int_0^1 (2x-1)f(x)dx\)
\(= 0 - [0 - 0] - \int_0^1 (2x-1)f(x)dx\)
\(= -\int_0^1 (2x-1)f(x)dx\)
但更直接的方法是使用分部积分：
\(\int_0^1 x(x-1)f''(x)dx = [x(x-1)f'(x)]_0^1 - \int_0^1 (2x-1)f'(x)dx\)
\(= 0 - [(2x-1)f(x)]_0^1 + \int_0^1 2f(x)dx\)
\(= -[0 - 0] + 2\int_0^1 f(x)dx = 2\int_0^1 f(x)dx\)
因此，\(\int_0^1 f(x)dx = -\int_0^1 x(x-1)f''(x)dx\)
证明(2)：
由(1)得：\(\int_0^1 f(x)dx = \int_0^1 x(x-1)f''(x)dx\)
由于\(x(x-1) \leq 0\)，\(x \in [0,1]\)，所以\(|x(x-1)| = x(1-x)\)
\(\int_0^1 x(x-1)f''(x)dx \leq \int_0^1 |x(x-1)f''(x)| dx\)
\(\leq \max f''(x) \int_0^1 x(1-x)dx = \max f''(x) \left[\frac{x^2}{2} - \frac{x^3}{3}\right]_0^1 = \max f''(x) \cdot \frac{1}{6}\)
但实际计算得：\(\int_0^1 x(1-x)dx = \frac{1}{6}\)
所以\(\int_0^1 f(x)dx \leq \frac{1}{6}\max f''(x)\)

2. 证明积分不等式

解题思路：

1. 分析被积函数的性质
2. 选择适当的积分方法（如积分中值定理、夹逼准则）
3. 利用已知条件进行推导
4. 通过恒等变形得出所需不等式

例11.5：设函数\(f(x) = x - [x]\)，其中\([x]\)表示不超过\(x\)的最大整数，则\(\lim_{x \to +\infty} \frac{1}{x}\int_0^x f(t)dt = \frac{1}{2}\)

解题步骤：

由于\(f(x)\)是周期为1的周期函数，其图像如图11-1所示
计算：\(\int_0^x f(t)dt = n\int_0^1 f(t)dt\)，其中\(n = [x]\)
每个周期内的面积为：\(\frac{1}{2}\)，所以\(\int_0^1 f(t)dt = \frac{1}{2}\)
当\(n \leq x < n+1\)时，有：\(\frac{n}{2} \leq \int_0^x f(t)dt < \frac{n+1}{2}\)
两边除以\(x\)，得：\(\frac{n/2}{x} \leq \frac{1}{x}\int_0^x f(t)dt < \frac{(n+1)/2}{x}\)
当\(x \to +\infty\)时，\(n \to \infty\)，\(\frac{n}{x} \to 1\)，\(\frac{n+1}{x} \to 1\)
由夹逼准则，得：\(\lim_{x \to +\infty} \frac{1}{x}\int_0^x f(t)dt = \frac{1}{2}\)

四、易错点与注意事项

1. 积分等式的证明

需要选择适当的积分方法，如分部积分法
注意边界条件的处理，如\(f(0) = f(1) = 0\)

2. 积分不等式的证明

要注意被积函数的符号
选择合适的不等式形式，如夹逼准则

3. 周期函数的处理

周期函数的积分可以利用周期性简化计算
注意积分区间的划分

4. 夹逼准则的应用

确保上下界都收敛到同一个值
不能只用一个不等式

5. 积分中值定理的使用

必须保证函数在区间上连续
不能直接应用在不连续的函数上

五、重要公式与定理

1. 重要极限：\(\lim_{t \to 0^+} t^a \ln t = 0 \quad (a > 0)\)
2. 积分中值定理：若\(f(x)\)在\([a,b]\)上连续，则存在\(\xi \in [a,b]\)，使\(\int_a^b f(x)dx = f(\xi)(b-a)\)
3. 夹逼准则：若\(f(x) \leq g(x) \leq h(x)\)，且\(\lim f(x) = \lim h(x) = A\)，则\(\lim g(x) = A\)
4. 分部积分法：\(\int u \, dv = uv - \int v \, du\)
5. 周期函数的积分性质：若\(f(x)\)是周期为\(T\)的周期函数，则\(\int_a^{a+T} f(x)dx = \int_0^T f(x)dx\)

六、典型例题解析

例1：设\(f(x)\)在\([0,1]\)上连续，证明\(\lim_{n \to \infty}\int_0^1 x^n f(x)dx = 0\)

解题思路：

由于\(f(x)\)在\([0,1]\)上连续，故有界，设\(|f(x)| \leq M\)
则：\(\left|\int_0^1 x^n f(x)dx\right| \leq \int_0^1 x^n |f(x)| dx \leq M\int_0^1 x^n dx = \frac{M}{n+1}\)
当\(n \to \infty\)时，\(\frac{M}{n+1} \to 0\)
由夹逼准则，得：\(\lim_{n \to \infty}\int_0^1 x^n f(x)dx = 0\)

例2：设\(f(x)\)在\([0,2]\)上具有一阶连续导数，且\(f'(x) \geq 0\)，证明：对任意正整数\(n\)有\(\int_0^{2\pi} f(x)\sin nx \, dx \leq \frac{f(2)-f(0)}{n}\)

解题思路：

用分部积分法：
\(\int_0^{2\pi} f(x)\sin nx \, dx = -\int_0^{2\pi} f(x)\frac{\cos nx}{n} dx\)
由于\(|\cos nx| \leq 1\)，所以：
\(\int_0^{2\pi} f(x)\sin nx \, dx \leq \frac{1}{n}\int_0^{2\pi} |f'(x)\cos nx| dx\)
由于\(f'(x) \geq 0\)，所以：
\(\leq \frac{1}{n}\int_0^{2\pi} f'(x)dx = \frac{1}{n}[f(2\pi)-f(0)]\)
因此：\(\int_0^{2\pi} f(x)\sin nx \, dx \leq \frac{f(2)-f(0)}{n}\)
