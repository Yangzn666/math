二、核心概念

1. 函数单调性的判定

若\(f'(x) \geq 0\)，\(a < x < b\)，则\(f(x)\)单调增加
若\(f'(x) \leq 0\)，\(a < x < b\)，则\(f(x)\)单调减少

2. 极值的判定

一阶导数判别法：

\(f'(x)\)在\(x_0\)左右变号：
由正变负：\(x_0\)为极大值点
由负变正：\(x_0\)为极小值点

二阶导数判别法：

\(f'(x_0) = 0\)，\(f''(x_0) \neq 0\)：
\(f''(x_0) < 0\)：\(x_0\)为极大值点
\(f''(x_0) > 0\)：\(x_0\)为极小值点

3. 中值定理

拉格朗日中值定理：

若\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导，则存在\(\xi \in (a,b)\)，使\(f(b)-f(a)=f'(\xi)(b-a)\)

积分中值定理：

若\(f(x)\)在\([a,b]\)上连续，则存在\(\xi \in [a,b]\)，使\(\int_a^b f(x)dx = f(\xi)(b-a)\)

三、常见题型及解题思路

1. 判断函数的单调性

解题思路：

1. 求函数的一阶导数\(f'(x)\)
2. 找出\(f'(x) = 0\)的点和\(f'(x)\)不存在的点（临界点）
3. 利用临界点将区间分成若干子区间
4. 判断每个子区间内\(f'(x)\)的符号
5. 确定函数的单调区间

例：判断\(f(x) = x^3 - 3x\)的单调性

求导：\(f'(x) = 3x^2 - 3 = 3(x^2 - 1)\)
找临界点：令\(f'(x) = 0\)，得\(x = -1, x = 1\)
区间划分：\((-\infty, -1), (-1, 1), (1, +\infty)\)
符号判断：
\(x < -1\)时，\(f'(x) > 0\)，函数单调增加
\(-1 < x < 1\)时，\(f'(x) < 0\)，函数单调减少
\(x > 1\)时，\(f'(x) > 0\)，函数单调增加

2. 求函数的极值

解题思路：

1. 求函数的一阶导数\(f'(x)\)
2. 找出\(f'(x) = 0\)的点和\(f'(x)\)不存在的点（临界点）
3. 判断临界点是否为极值点
    一阶导数判别法：判断\(f'(x)\)在临界点左右邻域的符号变化
    二阶导数判别法：计算\(f''(x)\)在临界点的值
4. 计算极值

例：求\(f(x) = x^3 - 3x\)的极值

求导：\(f'(x) = 3x^2 - 3\)
找临界点：令\(f'(x) = 0\)，得\(x = -1, x = 1\)
二阶导数：\(f''(x) = 6x\)
判断：\(f''(-1) = -6 < 0\)，故\(x = -1\)为极大值点，极大值为\(f(-1) = 2\)
判断：\(f''(1) = 6 > 0\)，故\(x = 1\)为极小值点，极小值为\(f(1) = -2\)

3. 应用中值定理证明不等式

解题思路：

1. 确定需要证明的不等式
2. 构造辅助函数
3. 验证辅助函数满足中值定理条件
4. 应用中值定理，得到所需不等式

例：证明当\(0 < a < b\)时，\(\ln b - \ln a < \frac{b - a}{a}\)

构造函数：设\(f(x) = \ln x\)，\(x > 0\)
验证条件：\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导
应用定理：由拉格朗日中值定理，存在\(\xi \in (a,b)\)，使\(f(b) - f(a) = f'(\xi)(b - a)\)
计算导数：\(f'(\xi) = \frac{1}{\xi}\)，且\(a < \xi < b\)，所以\(\frac{1}{b} < \frac{1}{\xi} < \frac{1}{a}\)
得出结论：因此，\(\ln b - \ln a = \frac{b - a}{\xi} < \frac{b - a}{a}\)

4. 利用单调性证明方程根的存在性

解题思路：

1. 确定函数\(f(x)\)
2. 分析\(f(x)\)的单调性
3. 计算\(f(x)\)在区间端点的函数值
4. 利用介值定理或单调性判断根的存在性

例：证明方程\(x^3 - 3x + 1 = 0\)在\((0,1)\)内有且仅有一个实根

构造函数：设\(f(x) = x^3 - 3x + 1\)
求导分析：\(f'(x) = 3x^2 - 3 = 3(x^2 - 1) < 0\)，当\(x \in (0,1)\)时
单调性：\(f(x)\)在\((0,1)\)上单调减少
端点值：\(f(0) = 1 > 0\)，\(f(1) = -1 < 0\)
存在性：由介值定理，存在\(\xi \in (0,1)\)，使\(f(\xi) = 0\)
唯一性：由于\(f(x)\)在\((0,1)\)上单调减少，故根唯一

四、易错点与注意事项

1. 单调性与极值的关系

单调区间和极值点是相关但不同的概念
函数在极值点处不一定是单调的

2. 中值定理的应用

需要验证函数在区间上满足中值定理的条件
不能直接应用中值定理，需要构造合适的辅助函数

3. 一阶导数与二阶导数的判别

一阶导数判别法适用于所有可导函数
二阶导数判别法仅适用于\(f'(x_0) = 0\)且\(f''(x_0) \neq 0\)的情况

4. 区间端点的考虑

在闭区间上考虑单调性时，需要考虑端点
极值可能在区间端点处取得

5. 导数不存在的点

导数不存在的点可能是极值点，但不能用二阶导数判别法
需要用一阶导数判别法或直接判断

五、重要公式与定理

1. 拉格朗日中值定理：若\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导，则存在\(\xi \in (a,b)\)，使\(f(b)-f(a)=f'(\xi)(b-a)\)
2. 积分中值定理：若\(f(x)\)在\([a,b]\)上连续，则存在\(\xi \in [a,b]\)，使\(\int_a^b f(x)dx = f(\xi)(b-a)\)
3. 单调性判定：
    \(f'(x) \geq 0 \Rightarrow f(x)\)单调增加
    \(f'(x) \leq 0 \Rightarrow f(x)\)单调减少
4. 极值判定：
    一阶导数判别法：\(f'(x)\)在\(x_0\)左右变号
    二阶导数判别法：\(f'(x_0) = 0\)，\(f''(x_0) \neq 0\)
5. 方程根的存在性：
    若\(f(x)\)在\([a,b]\)上连续，\(f(a)f(b) < 0\)，则至少存在一个\(\xi \in (a,b)\)，使\(f(\xi) = 0\)
    若\(f(x)\)在\([a,b]\)上单调，则根唯一
