二、核心概念

1. 二重积分的定义

定义：设\(f(x, y)\)是有界闭区域\(D\)上的有界函数，将闭区域\(D\)任意分成\(n\)个小闭区域\(\Delta\sigma_1, \Delta\sigma_2, \ldots, \Delta\sigma_n\)，其中\(\Delta\sigma_i\)表示第\(i\)个小闭区域，也表示它的面积。在每个\(\Delta\sigma_i\)上任取一点\((\xi_i, \eta_i)\)，作乘积\(f(\xi_i, \eta_i)\Delta\sigma_i(i=1, 2, \ldots, n)\)，并作和\(\sum f(\xi_i, \eta_i)\Delta\sigma_i\)。如果当各小闭区域的直径中的最大值\(\lambda\)趋于零时，和的极限总存在（与\(\Delta\sigma_i\)的分法及点\((\xi_i, \eta_i)\)的取法均无关），则称此极限值为函数\(f(x, y)\)在闭区域\(D\)上的二重积分，记作\(\iint_D f(x, y)d\sigma\)，即

\(\iint_D f(x, y)d\sigma = \lim_{\lambda \to 0} \sum f(\xi_i, \eta_i)\Delta\sigma_i\)

几何意义：

若\(f(x, y) \geq 0\)，则二重积分表示以\(D\)为底，曲面\(z = f(x, y)\)为顶的曲顶柱体的体积
若\(f(x, y) < 0\)，则二重积分的绝对值表示体积，但二重积分值为负

2. 二重积分的性质

1. 线性性质：\(\iint_D [kf(x, y) + g(x, y)]d\sigma = k\iint_D f(x, y)d\sigma + \iint_D g(x, y)d\sigma\)
2. 区域可加性：若\(D = D_1 \cup D_2\)，且\(D_1\)与\(D_2\)的公共部分的面积为0，则
    \(\iint_D f(x, y)d\sigma = \iint_{D_1} f(x, y)d\sigma + \iint_{D_2} f(x, y)d\sigma\)
3. 积分中值定理：若\(f(x, y)\)在\(D\)上连续，则存在点\((\xi, \eta) \in D\)，使
    \(\iint_D f(x, y)d\sigma = f(\xi, \eta) \cdot A\)，其中\(A\)为\(D\)的面积
4. 保号性：若\(f(x, y) \geq 0\)，则\(\iint_D f(x, y)d\sigma \geq 0\)

3. 二重积分的对称性

普通对称性：

若\(D\)关于\(x\)轴对称，\(f(x, y) = f(x, -y)\)，则\(\iint_D f(x, y)d\sigma = 2\iint_{D_1} f(x, y)d\sigma\)，其中\(D_1\)为\(D\)在\(x\)轴上方的部分
若\(D\)关于\(y\)轴对称，\(f(x, y) = f(-x, y)\)，则\(\iint_D f(x, y)d\sigma = 2\iint_{D_2} f(x, y)d\sigma\)，其中\(D_2\)为\(D\)在\(y\)轴右侧的部分

轮换对称性：

若\(D\)关于\(y = x\)对称，\(f(x, y) = f(y, x)\)，则\(\iint_D f(x, y)d\sigma = \iint_D f(y, x)d\sigma\)

三、常见题型及解题思路

1. 直角坐标系下二重积分的计算

解题思路：

1. 确定积分区域\(D\)的边界
2. 判断先积\(x\)还是先积\(y\)
3. 确定积分上下限
4. 计算积分

例：计算\(\iint_D (x + y)d\sigma\)，其中\(D\)由\(x = 0, x = 1, y = 0, y = 1\)围成

先积\(y\)：\(\int_0^1\int_0^1 (x + y)dydx = \int_0^1[xy + \frac{y^2}{2}]_0^1dx = \int_0^1(x + \frac{1}{2})dx = [\frac{x^2}{2} + \frac{x}{2}]_0^1 = 1\)
先积\(x\)：\(\int_0^1\int_0^1 (x + y)dxdy = \int_0^1[\frac{x^2}{2} + xy]_0^1dy = \int_0^1(\frac{1}{2} + y)dy = [\frac{y}{2} + \frac{y^2}{2}]_0^1 = 1\)

2. 极坐标系下二重积分的计算

解题思路：

1. 确定积分区域\(D\)的边界（用极坐标表示）
2. 判断先积\(r\)还是先积\(\theta\)
3. 确定积分上下限
4. 计算积分

例：计算\(\iint_D e^{x^2+y^2}d\sigma\)，其中\(D\)为\(x^2 + y^2 \leq 1\)

极坐标变换：\(x = r\cos\theta, y = r\sin\theta, d\sigma = r dr d\theta\)
积分区域：\(0 \leq r \leq 1, 0 \leq \theta \leq 2\pi\)
积分：\(\int_0^{2\pi}\int_0^1 e^{r^2} \cdot r dr d\theta = 2\pi\int_0^1 e^{r^2} \cdot r dr = 2\pi \cdot [\frac{e^{r^2}}{2}]_0^1 = \pi(e - 1)\)

3. 利用对称性简化二重积分

解题思路：

1. 判断积分区域\(D\)是否具有对称性
2. 判断被积函数是否具有对称性
3. 利用对称性将积分区域缩小，简化计算

例：计算\(\iint_D (x^2 + y^2 + xy)d\sigma\)，其中\(D\)为\(x^2 + y^2 \leq 1\)

利用对称性：\(D\)关于\(x\)轴和\(y\)轴对称
\(f(x, y) = x^2 + y^2 + xy\)，其中\(x^2 + y^2\)关于\(x\)轴和\(y\)轴对称，\(xy\)关于\(x\)轴和\(y\)轴不对称
\(\iint_D xy d\sigma = 0\)（因为\(xy\)关于\(x\)轴和\(y\)轴对称，但符号相反）
\(\iint_D (x^2 + y^2) d\sigma = 2\iint_{D_1} (x^2 + y^2) d\sigma\)，其中\(D_1\)为\(D\)在第一象限的部分
极坐标计算：\(\iint_D (x^2 + y^2) d\sigma = \int_0^{2\pi}\int_0^1 r^2 \cdot r dr d\theta = 2\pi\int_0^1 r^3 dr = 2\pi \cdot [\frac{r^4}{4}]_0^1 = \frac{\pi}{2}\)

四、易错点与注意事项

1. 直角坐标系下积分次序的选择

先积\(x\)还是先积\(y\)，取决于区域\(D\)的形状和被积函数
通常先积较简单的变量

2. 极坐标系下积分次序的选择

通常先积\(r\)，再积\(\theta\)
极坐标系下，\(r \geq 0\)，\(\theta \in [0, 2\pi)\)

3. 极坐标与直角坐标的转换

\(x = r\cos\theta, y = r\sin\theta\)
\(x^2 + y^2 = r^2\)
\(d\sigma = r dr d\theta\)

4. 对称性问题

对称性尽量用对称方法去解决，避免直接计算
注意被积函数的奇偶性

5. 积分区域的确定

画图确定积分区域，避免上下限错误
分段区域需要分区域计算

五、重要公式汇总

1. 直角坐标系下的二重积分：
    \(\iint_D f(x, y)d\sigma = \int_a^b\int_{g_1(x)}^{g_2(x)} f(x, y)dydx\)
    \(\iint_D f(x, y)d\sigma = \int_c^d\int_{h_1(y)}^{h_2(y)} f(x, y)dxdy\)
2. 极坐标系下的二重积分：
    \(\iint_D f(x, y)d\sigma = \int_\alpha^\beta\int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta, r\sin\theta) \cdot r dr d\theta\)
3. 极坐标与直角坐标的转换：
    \(x = r\cos\theta, y = r\sin\theta\)
    \(x^2 + y^2 = r^2\)
    \(d\sigma = r dr d\theta\)
4. 对称性应用：
    若\(D\)关于\(x\)轴对称，\(f(x, y) = f(x, -y)\)，则\(\iint_D f(x, y)d\sigma = 2\iint_{D_1} f(x, y)d\sigma\)
    若\(D\)关于\(y\)轴对称，\(f(x, y) = f(-x, y)\)，则\(\iint_D f(x, y)d\sigma = 2\iint_{D_2} f(x, y)d\sigma\)

六、典型例题解析

例1：计算\(\iint_D (x^2 + y^2) d\sigma\)，其中\(D\)为\(x^2 + y^2 \leq 2x\)

解题思路：

将\(D\)化为极坐标：\(x^2 + y^2 \leq 2x \Rightarrow r^2 \leq 2r\cos\theta \Rightarrow r \leq 2\cos\theta\)
确定\(\theta\)的范围：\(r = 2\cos\theta \geq 0 \Rightarrow \cos\theta \geq 0 \Rightarrow \theta \in [-\frac{\pi}{2}, \frac{\pi}{2}]\)
积分：\(\iint_D (x^2 + y^2) d\sigma = \int_{-\pi/2}^{\pi/2}\int_0^{2\cos\theta} r^2 \cdot r dr d\theta\)
计算：\(\int_{-\pi/2}^{\pi/2}\int_0^{2\cos\theta} r^3 dr d\theta = \int_{-\pi/2}^{\pi/2} [\frac{r^4}{4}]_0^{2\cos\theta} d\theta = \int_{-\pi/2}^{\pi/2} 4\cos^4\theta d\theta\)
利用三角恒等式：\(\cos^4\theta = \frac{(1 + \cos2\theta)^2}{4} = \frac{1 + 2\cos2\theta + \cos^22\theta}{4} = \frac{1 + 2\cos2\theta + \frac{1 + \cos4\theta}{2}}{4} = \frac{3 + 4\cos2\theta + \cos4\theta}{8}\)
积分结果：\(4\int_{-\pi/2}^{\pi/2} \frac{3 + 4\cos2\theta + \cos4\theta}{8} d\theta = \frac{1}{2}[3\theta + 2\sin2\theta + \frac{1}{4}\sin4\theta]_{-\pi/2}^{\pi/2} = \frac{1}{2}[3\pi] = \frac{3\pi}{2}\)

例2：计算\(\iint_D e^{x^2+y^2} d\sigma\)，其中\(D\)为\(x^2 + y^2 \leq 4\)

解题思路：

极坐标变换：\(x = r\cos\theta, y = r\sin\theta, d\sigma = r dr d\theta\)
积分区域：\(0 \leq r \leq 2, 0 \leq \theta \leq 2\pi\)
积分：\(\int_0^{2\pi}\int_0^2 e^{r^2} \cdot r dr d\theta\)
计算：\(\int_0^{2\pi} d\theta \cdot \int_0^2 r e^{r^2} dr = 2\pi \cdot [\frac{e^{r^2}}{2}]_0^2 = \pi(e^4 - 1)\)
