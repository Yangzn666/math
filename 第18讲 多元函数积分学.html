二、核心概念

1. 第二型曲面积分

定义：\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy\)

物理意义：表示向量场\((P, Q, R)\)穿过曲面\(\Sigma\)的通量

有向曲面：曲面有方向，法向量指向外侧或内侧

2. 曲面投影与投影面积

投影到xOy面：\(dxdy = \cos\gamma dS\)，其中\(\gamma\)为曲面法向量与z轴的夹角

当法向量与z轴正向夹角为锐角时，\(\cos\gamma > 0\)，\(dxdy = \cos\gamma dS\)
当法向量与z轴正向夹角为钝角时，\(\cos\gamma < 0\)，\(dxdy = -\cos\gamma dS\)

投影到yOz面：\(dydz = \cos\alpha dS\)，其中\(\alpha\)为曲面法向量与x轴的夹角

投影到xOz面：\(dzdx = \cos\beta dS\)，其中\(\beta\)为曲面法向量与y轴的夹角

3. 曲面积分的计算

投影到xOy面：

\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy = \iint_D [P(-\frac{\partial z}{\partial x}) + Q(-\frac{\partial z}{\partial y}) + R] dxdy\)
其中D为\(\Sigma\)在xOy面上的投影，\(z = z(x, y)\)

投影到yOz面：

\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy = \iint_D [P + Q(-\frac{\partial x}{\partial z}) + R(-\frac{\partial x}{\partial y})] dydz\)
其中D为\(\Sigma\)在yOz面上的投影，\(x = x(y, z)\)

投影到xOz面：

\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy = \iint_D [P(-\frac{\partial y}{\partial z}) + Q + R(-\frac{\partial y}{\partial x})] dzdx\)
其中D为\(\Sigma\)在xOz面上的投影，\(y = y(x, z)\)

4. 特殊曲面的面积元素

柱面：\(x^2 + y^2 = a^2\)，\(dS = a dxdz\)
球面：\(x^2 + y^2 + z^2 = a^2\)，\(dS = a dxdy / x\)
锥面：\(z = \sqrt{x^2 + y^2}\)，\(dS = \sqrt{2} dxdy\)

三、常见题型及解题思路

1. 第二型曲面积分的计算

解题思路：

1. 确定曲面\(\Sigma\)的方程和方向（外侧或内侧）
2. 选择合适的投影面（xOy面、yOz面或xOz面）
3. 将曲面积分转化为二重积分
4. 计算二重积分

例18.22：计算\(I = \iint_\Sigma 2e^{2\sqrt{x^2+y^2}} dxdy\)，其中\(\Sigma\)是由曲线L绕z轴旋转一周得到的曲面，L过点A(-1,0,1)与B(0,0,0)，且\(\Sigma\)被z=1，z=2所围有界闭区域的边界曲面，取外侧

解题思路：

1. 确定曲面\(\Sigma\)：L的方程为x = -z, y = 0，绕z轴旋转得到\(\Sigma\)：\(x^2 + y^2 = z^2\)
2. 将\(\Sigma\)分成三部分：\(\Sigma_1\)（z=1, \(x^2+y^2\leq1\)）、\(\Sigma_2\)（z=2, \(x^2+y^2\leq4\)）、\(\Sigma_3\)（z=\(\sqrt{x^2+y^2}\), \(1\leq z\leq2\)）
3. 分别计算各部分的积分：
    \(\Sigma_1\)：\(\iint_{\Sigma_1} 2e^{2\sqrt{x^2+y^2}} dxdy = 0\)（因为z=1，dxdy = 0）
    \(\Sigma_2\)：\(\iint_{\Sigma_2} 2e^{2\sqrt{x^2+y^2}} dxdy = 2\int_0^{2\pi}\int_0^2 e^{2r} r dr d\theta = 4\pi(e^4 - 1)/4 = \pi(e^4 - 1)\)
    \(\Sigma_3\)：\(\iint_{\Sigma_3} 2e^{2\sqrt{x^2+y^2}} dxdy = 2\int_0^{2\pi}\int_1^2 e^{2r} r dr d\theta = 4\pi(e^4 - e^2)/4 = \pi(e^4 - e^2)\)
4. 总和：\(I = 0 + \pi(e^4 - 1) + \pi(e^4 - e^2) = \pi(2e^4 - e^2 - 1)\)

2. 挖洞法（包围奇点的曲面积分）

解题思路：

1. 当积分区域包含奇点时，作一包含奇点的球面
2. 利用高斯公式计算包围奇点的曲面积分
3. 将原积分转化为包围奇点的曲面积分

例18.28：设\(\Sigma\)是椭球面\(\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1\)，法向量指向外侧，计算\(\iint_\Sigma \frac{xdydz + ydzdx + zdxdy}{(x^2+y^2+z^2)^{3/2}}\)

解题思路：

1. 积分区域包含奇点(0,0,0)，在\(\Sigma\)内部作一球面\(\Sigma_1\)：\(x^2+y^2+z^2=r^2\)（r>0），法向量指向外侧
2. 由高斯公式：
    \(\iint_\Sigma \frac{xdydz + ydzdx + zdxdy}{(x^2+y^2+z^2)^{3/2}} = \iint_{\Sigma_1} \frac{xdydz + ydzdx + zdxdy}{(x^2+y^2+z^2)^{3/2}}\)
3. 在\(\Sigma_1\)上，\(x^2+y^2+z^2=r^2\)，所以：
    \(\iint_{\Sigma_1} \frac{xdydz + ydzdx + zdxdy}{(x^2+y^2+z^2)^{3/2}} = \frac{1}{r^3}\iint_{\Sigma_1} xdydz + ydzdx + zdxdy\)
4. 由对称性，\(\iint_{\Sigma_1} xdydz = \iint_{\Sigma_1} ydzdx = \iint_{\Sigma_1} zdxdy = \frac{4\pi r^2}{3}\)
5. 所以，\(\iint_{\Sigma_1} \frac{xdydz + ydzdx + zdxdy}{(x^2+y^2+z^2)^{3/2}} = \frac{1}{r^3} \cdot \frac{4\pi r^2}{3} \cdot 3 = 4\pi\)
6. 因此，原积分结果为\(4\pi\)

3. 选择合适的投影面

解题思路：

1. 分析曲面的方程，选择投影面使计算简便
2. 当曲面在某一坐标面上的投影为曲线时，不能选择该坐标面投影
3. 选择投影面时，考虑曲面的形状和被积函数

例18.13：设\(\Sigma\)为椭球面\(\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1\)的上半部分，P(x,y,z)为\(\Sigma\)在点P处的切平面，p(x,y,z)为点O(0,0,0)到平面的距离，求\(\iint_\Sigma \frac{z dS}{p(x,y,z)}\)

解题思路：

1. 椭球面的切平面方程：\(\frac{x_0}{a^2}(x-x_0) + \frac{y_0}{b^2}(y-y_0) + \frac{z_0}{c^2}(z-z_0) = 0\)
2. 点O(0,0,0)到切平面的距离：
    \(p(x_0,y_0,z_0) = - \frac{x_0^2/a^2 + y_0^2/b^2 + z_0^2/c^2}{\sqrt{(x_0/a^2)^2 + (y_0/b^2)^2 + (z_0/c^2)^2}} = \frac{1}{\sqrt{(x_0/a^2)^2 + (y_0/b^2)^2 + (z_0/c^2)^2}}\)
3. 所以，\(\frac{z}{p(x,y,z)} = z \sqrt{(x/a^2)^2 + (y/b^2)^2 + (z/c^2)^2}\)
4. 由于\(\Sigma\)为椭球面上半部分，\(z = c\sqrt{1 - x^2/a^2 - y^2/b^2}\)，可以计算积分

四、易错点与注意事项

1. 投影面的选择

当曲面在某一坐标面上的投影为曲线时，不能选择该坐标面投影
例如，当曲面为球面时，投影到xOy面时，投影为圆，可以计算；但若曲面为z轴，投影到xOy面时，投影为曲线，不能计算

2. 有向曲面的投影符号

当曲面的法向量与坐标轴正向夹角为锐角时，投影为正；为钝角时，投影为负
例如，对于z = f(x,y)，当\(f_z < 0\)时，法向量与z轴正向夹角为钝角，投影为负

3. 特殊曲面的面积元素

柱面、球面、锥面的面积元素公式要记清楚
例如，柱面\(x^2+y^2=a^2\)的\(dS = a dxdz\)，而不是a dxdy

4. 挖洞法的适用条件

当积分区域包含奇点时，可以使用挖洞法
挖洞后，包围奇点的曲面积分与原曲面积分相等

5. 曲面方程的化简

曲面方程可能需要化简为显式形式z = f(x,y)才能计算
例如，椭球面\(\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1\)的上半部分可化为\(z = c\sqrt{1 - x^2/a^2 - y^2/b^2}\)

五、重要公式汇总

1. 第二型曲面积分的计算：
    投影到xOy面：\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy = \iint_D [P(-\frac{\partial z}{\partial x}) + Q(-\frac{\partial z}{\partial y}) + R] dxdy\)
    投影到yOz面：\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy = \iint_D [P + Q(-\frac{\partial x}{\partial z}) + R(-\frac{\partial x}{\partial y})] dydz\)
    投影到xOz面：\(\iint_\Sigma Pdydz + Qdzdx + Rdxdy = \iint_D [P(-\frac{\partial y}{\partial z}) + Q + R(-\frac{\partial y}{\partial x})] dzdx\)
2. 特殊曲面的面积元素：
    柱面\(x^2+y^2=a^2\)：\(dS = a dxdz\)
    球面\(x^2+y^2+z^2=a^2\)：\(dS = a dxdy / x\)
    锥面\(z=\sqrt{x^2+y^2}\)：\(dS = \sqrt{2} dxdy\)
3. 挖洞法：
    当积分区域包含奇点时，可以在奇点周围作一球面，利用高斯公式计算
    \(\iint_\Sigma F\cdot dS = \iint_{\Sigma_1} F\cdot dS\)，其中\(\Sigma_1\)是包围奇点的球面

六、典型例题解析

例1：计算\(\iint_\Sigma (x^2+y^2+z^2) dS\)，其中\(\Sigma\)为球面\(x^2+y^2+z^2=a^2\)的上半部分

解题思路：

1. 球面方程：\(x^2+y^2+z^2=a^2\)，上半部分\(z = \sqrt{a^2-x^2-y^2}\)
2. 计算面积元素：\(dS = \sqrt{1 + (z_x)^2 + (z_y)^2} dxdy = \sqrt{1 + (x^2+z^2)/z^2 + (y^2+z^2)/z^2} dxdy = a/z dxdy\)
3. 代入积分：
    \(\iint_\Sigma (x^2+y^2+z^2) dS = \iint_D a^2 \cdot (a/z) dxdy = a^3 \iint_D 1/z dxdy\)
4. 其中D为\(x^2+y^2\leq a^2\)，\(z = \sqrt{a^2-x^2-y^2}\)，所以：
    \(= a^3 \iint_D 1/\sqrt{a^2-x^2-y^2} dxdy\)
5. 极坐标变换：\(x = r\cos\theta, y = r\sin\theta\)
    \(= a^3 \int_0^{2\pi}\int_0^a 1/\sqrt{a^2-r^2} \cdot r dr d\theta\)
    \(= 2\pi a^3 \int_0^a r/\sqrt{a^2-r^2} dr\)
    \(= 2\pi a^3 [ -\sqrt{a^2-r^2} ]_0^a = 2\pi a^3 \cdot a = 2\pi a^4\)

例2：计算\(\iint_\Sigma y dxdy\)，其中\(\Sigma\)为曲面\(z = x^2 + y^2\)（\(0\leq z\leq 1\)）的上侧

解题思路：

1. 曲面方程：\(z = x^2 + y^2\)，上侧
2. 投影到xOy面，\(dxdy = \cos\gamma dS\)，其中\(\gamma\)为法向量与z轴的夹角
3. 法向量：\((-2x, -2y, 1)\)，所以\(\cos\gamma = 1/\sqrt{4x^2+4y^2+1}\)
4. 但第二型曲面积分中，y dxdy = y dxdy（不考虑dS）
5. 所以，\(\iint_\Sigma y dxdy = \iint_D y dxdy\)，其中D为\(x^2+y^2\leq1\)
6. 由对称性，\(\iint_D y dxdy = 0\)（y关于x轴对称，y为奇函数）
