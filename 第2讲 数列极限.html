二、核心概念

1. 数列的概念

数列是特殊的函数，可理解为自变量为正整数\(n\)的函数\(x=f(n)\)，\(n \in \mathbb{N^+}\)

常见数列类型：等差数列、等比数列、单调数列、有界数列

子列：从原数列中选取无穷多项按原顺序组成的新数列

2. 数列极限的定义

设\(\{x_n\}\)为一数列，若存在常数\(a\)，对任意\(\varepsilon>0\)，总存在正整数\(N\)，当\(n>N\)时，\( |x_n-a|<\varepsilon \)恒成立，则称\(a\)是数列\(\{x_n\}\)的极限，记为\(\lim x_n = a\)

3. 数列极限与函数极限的区别

唯一性：相同
有界性：数列极限是全局有界，函数极限是局部有界
保号性：数列极限强调存在\(N\)，当\(n>N\)时才有\(x_n>0\)(或\(<0\))

三、数列极限存在的两个准则

1. 夹逼准则

条件：从某项起，\(y_n \leq x_n \leq z_n\)，且\(\lim y_n = \lim z_n = a\)
结论：\(\lim x_n = a\)

常用放缩方法：

分母放大(分数变小)，分母缩小(分数变大)
利用重要不等式：\(\sin x0)\)、\(e^x≥x+1\)、\(x-1≥\ln x\ (x>0)\)等
利用闭区间上连续函数的最大值与最小值

2. 单调有界准则

条件：数列\(\{x_n\}\)单调增加(减少)且有上界(下界)
结论：\(\lim x_n\)存在

证明步骤：

1. 证明数列单调：作差法(\(x_{n+1}-x_n\))、数学归纳法、利用函数单调性
2. 证明数列有界：找\(M\)使\(|x_n| \leq M\)、放缩法、基本不等式法

四、海涅定理(归结原则)

1. 定理内容

设\(f(x)\)在\(U(x_0,\delta)\)内有定义，则\(\lim f(x) = A\)存在→对任何\(U(x_0,\delta)\)内以\(x_0\)为极限的数列\(\{x_n\}(x_n eq x_0)\)，极限\(\lim f(x_n) = A\)存在

常用转化：

\(x \to 0\)时，取\(x=\frac{1}{n} \to \lim f\left(\frac{1}{n}\right)=A\)
\(x \to +\infty\)时，取\(x=n \to \lim f(n)=A\)

2. 应用价值

将数列极限转化为函数极限，便于使用洛必达法则、泰勒公式、等价无穷小代换等工具

例：\(\lim\left(\frac{\ln n}{n}\right)=\lim\left(\frac{\ln x}{x}\right)=0 \quad (x \to +\infty)\)

五、收敛数列的性质

1. 基本性质

唯一性：若极限存在，唯一
有界性：若极限存在，数列有界
保号性：若\(\lim x_n = a > 0\)，则存在\(N>0\)，当\(n>N\)时，\(x_n>0\)

2. 极限四则运算

\(\lim(x_n \pm y_n) = \lim x_n \pm \lim y_n\)
\(\lim(x_n y_n) = \lim x_n \cdot \lim y_n\)
若\(\lim y_n \neq 0\)，则\(\lim\left(\frac{x_n}{y_n}\right) = \frac{\lim x_n}{\lim y_n}\)

六、证明数列极限存在性的思路

1. 单调有界准则应用思路

证明单调性：

作差法：计算\(x_{n+1}-x_n\)，判断符号
数学归纳法：验证\(n=1\)成立，假设\(n=k\)成立，证明\(n=k+1\)成立
利用函数单调性：若\(x_{n+1}=f(x_n)\)，\(f(x)\)单调，则\(\{x\}\)单调

证明有界性：

利用基本不等式放缩
找到一个合适的\(M\)，证明\(|x_n| \leq M\)
利用递推关系式找界

2. 夹逼准则应用思路

确定上下界：

分子不变，分母放大(分数变小)或缩小(分数变大)
利用已知不等式(如\(\sin x

构造夹逼数列：

找到两个极限相同的数列\(\{y_n\}\)和\(\{z_n\}\)，使得\(y_n \leq x_n \leq z_n\)

3. 海涅定理应用思路

1. 将数列极限转化为函数极限
2. 使用函数极限的计算方法(洛必达法则、泰勒展开等)
3. 由函数极限结果得到数列极限

七、常见题型及解题技巧

1. 证明数列极限存在性

思路：先证明数列有界且单调(单调有界准则)或满足夹逼条件(夹逼准则)
技巧：利用递推式找界，利用不等式放缩

2. 求解数列极限

思路：若无法直接求解，考虑：
1. 海涅定理转化成函数极限
2. 夹逼准则
3. 利用已知极限结论(如\(\lim\left(1+\frac{1}{n}\right)^n=e\))

3. 判断数列收敛性

思路：
若能找到两个收敛到不同极限的子列，则原数列发散
若能找到一个发散的子列，则原数列发散

4. 利用数列极限性质

思路：利用唯一性、有界性、保号性等性质判断数列行为

八、易错点与注意事项

1. 数列极限与函数极限的区别

数列极限是离散的，不能直接使用洛必达法则
有界性：数列是全局有界，函数是局部有界

2. 单调有界准则的使用

单调增加需证明有上界，单调减少需证明有下界
证明单调性时，要严格判断\(x_{n+1}-x_n\)的符号

3. 夹逼准则的放缩

放缩要适当，不能过宽(无法夹逼)或过窄(无法证明)
分母越大，分数越小；分子越大，分数越大

4. 海涅定理的使用

适用于无法直接求解的数列极限
转化后需验证函数极限存在

九、重要不等式与结论

1. \(1-\cos x \sim \frac{x^2}{2} \quad (x \to 0)\)
2. \(\sin x < x \quad (x > 0)\)
3. \(e^x \geq x+1 \quad (\text{任意}x)\)
4. \(x-1 \geq \ln x \quad (x > 0)\)
5. \(1+x > \ln(1+x) > \frac{x}{1+x} \quad (x > 0)\)
6. \(\lim\left(1+\frac{1}{n}\right)^n = e\)
7. \(\lim(n^{1/n}) = 1\)
