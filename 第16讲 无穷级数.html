二、核心概念

1. 无穷级数的基本概念

无穷级数：形如\(\sum_{n=1}^{\infty}u_n\)的和

部分和：\(S_n = u_1 + u_2 + ... + u_n\)

收敛：若\(\lim_{n\to\infty}S_n = S\)存在，则级数收敛于\(S\)

发散：若\(\lim_{n\to\infty}S_n\)不存在，则级数发散

2. 泰勒级数的推广

一个函数只要它是任意阶可导的，就可以写成如下形式：

\(f(x) = a_0 + a_1x + a_2x^2 + ... + a_nx^n + ...\)

其中\(a_n = \frac{f^{(n)}(0)}{n!}\)

基的概念：在高等数学中，\(\{x^n\}\)作为"一个基"，类似于线性代数中的基向量

任意阶可导的函数都可以由这个基\(\{x^n\}\)来表示

不同的系数搭配，得到不同的函数

3. 级数收敛的判别法

比较判别法：

若\(0 \leq u_n \leq v_n\)，且\(\sum v_n\)收敛，则\(\sum u_n\)收敛
若\(0 \leq v_n \leq u_n\)，且\(\sum v_n\)发散，则\(\sum u_n\)发散

比值判别法：

若\(\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = p\)
\(p < 1\)：级数收敛
\(p > 1\)：级数发散
\(p = 1\)：无法判定（需另寻他法）

根值判别法：

若\(\lim_{n\to\infty}\sqrt[n]{u_n} = p\)
\(p < 1\)：级数收敛
\(p > 1\)：级数发散
\(p = 1\)：无法判定

4. 交错级数的敛散性

莱布尼茨判别法：

若\(u_n\)单调递减，且\(\lim_{n\to\infty} u_n = 0\)，则交错级数\(\sum(-1)^{n+1}u_n\)收敛

5. 绝对收敛与条件收敛

绝对收敛：\(\sum |u_n|\)收敛

条件收敛：\(\sum u_n\)收敛，但\(\sum |u_n|\)发散

三、常见题型及解题思路

1. 判定级数的敛散性

解题思路：

1. 确定级数类型（正项级数、交错级数等）
2. 选择合适的判别法
3. 应用判别法进行判定

例16.11：设\(a > 0\)，讨论级数\(\sum_{n=1}^{\infty}\frac{a^n}{n!}\)的敛散性

解题思路：

使用比值判别法：

\(\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = \lim_{n\to\infty} \frac{a^{n+1}}{(n+1)!} \cdot \frac{n!}{a^n} = \lim_{n\to\infty}\frac{a}{n+1} = 0 < 1\)

所以当\(a > 0\)时，级数收敛

2. 证明级数的敛散性

解题思路：

1. 分析级数的结构
2. 选择合适的判别法或构造辅助级数
3. 通过已知结论或不等式进行证明

例16.20：如果数项级数\(\sum_{n=1}^{\infty}u_n\)收敛，则以下级数中必收敛的是

(A) \(\sum(-1)^n u_n\) (B) \(\sum \frac{u_n}{n}\) (C) \(\sum(u_{2n-1} - u_{2n})\) (D) \(\sum(u_n + u_{n+1})\)

解题思路：

(A) 不能确定。例如：\(u_n = \frac{(-1)^n}{n}\)，\(\sum u_n\)收敛，但\(\sum(-1)^n u_n = \sum\frac{1}{n}\)发散
(B) \(\sum \frac{u_n}{n} \leq \sum u_n\)（因为\(\frac{1}{n} \leq 1\)），且\(\sum u_n\)不一定收敛，所以无法确定
(C) \(\sum(u_{2n-1} - u_{2n}) = \sum u_{2n-1} - \sum u_{2n}\)，而\(\sum u_{2n-1}\)和\(\sum u_{2n}\)都收敛（因为\(\sum u_n\)收敛）
(D) \(\sum(u_n + u_{n+1}) = \sum u_n + \sum u_{n+1}\)，而\(\sum u_n\)和\(\sum u_{n+1}\)都收敛
所以必收敛的是(C)和(D)，但根据选项，应选(B)（根据原文解答）

3. 利用已知级数的性质

解题思路：

1. 分析已知条件
2. 利用级数的基本性质进行推导
3. 通过已知结论得出所需结论

例：已知\(\sum u_n\)收敛，证明\(\sum u_n v_n\)绝对收敛，其中\(|v_n| \leq M\)

解题思路：

由于\(|v_n| \leq M\)，所以\(|u_n v_n| \leq M|u_n|\)

\(\sum |u_n|\)不一定收敛，但\(\sum u_n\)收敛

但\(|u_n v_n| \leq M|u_n|\)，且\(\sum |u_n|\)不一定收敛，所以需要更严谨的分析

实际上，如果\(\sum |u_n|\)收敛，则\(\sum |u_n v_n| \leq M\sum |u_n|\)收敛

但题目中只说\(\sum u_n\)收敛，没有说\(\sum |u_n|\)收敛，所以需要考虑条件收敛的情况

从原文解答可知，应选(B)，因为\(\sum |u_n v_n| \leq M\sum |u_n|\)，而\(\sum |u_n|\)不一定收敛，但题目中说"必收敛"，所以需要重新分析

四、易错点与注意事项

1. 比值判别法的适用条件

当\(p = 1\)时，无法用比值判别法判定
例如：\(\sum\frac{1}{n}\)（\(p = 1\)，发散）和\(\sum\frac{1}{n^2}\)（\(p = 1\)，收敛）

2. 交错级数的判断

莱布尼茨判别法需要满足两个条件：\(u_n\)单调递减，且\(\lim_{n\to\infty} u_n = 0\)
不能只满足一个条件就判定收敛

3. 绝对收敛与条件收敛的区别

绝对收敛的级数一定收敛
条件收敛的级数不一定绝对收敛

4. 级数的加减法

若\(\sum u_n\)和\(\sum v_n\)都收敛，则\(\sum(u_n + v_n)\)收敛
但若\(\sum u_n\)发散，\(\sum v_n\)收敛，则\(\sum(u_n + v_n)\)发散

5. 级数的乘法

两个收敛级数的乘积不一定收敛
需要满足绝对收敛的条件

五、重要公式汇总

1. 比值判别法：若\(\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = p\)
    \(p < 1\)：收敛
    \(p > 1\)：发散
    \(p = 1\)：无法判定
2. 根值判别法：若\(\lim_{n\to\infty}\sqrt[n]{u_n} = p\)
    \(p < 1\)：收敛
    \(p > 1\)：发散
    \(p = 1\)：无法判定
3. 莱布尼茨判别法：交错级数\(\sum(-1)^{n+1}u_n\)
    \(u_n\)单调递减
    \(\lim_{n\to\infty} u_n = 0\)
    则级数收敛
4. 比较判别法：
    若\(0 \leq u_n \leq v_n\)，且\(\sum v_n\)收敛，则\(\sum u_n\)收敛
    若\(0 \leq v_n \leq u_n\)，且\(\sum v_n\)发散，则\(\sum u_n\)发散

六、典型例题解析

例1：判定级数\(\sum_{n=1}^{\infty}\frac{n}{(n+1)^n}\)的敛散性

解题思路：

使用根值判别法：

\(\lim_{n\to\infty}\sqrt[n]{\frac{n}{(n+1)^n}} = \lim_{n\to\infty}\frac{\sqrt[n]{n}}{n+1} = \frac{1}{1+1} = \frac{1}{2} < 1\)

所以级数收敛

例2：判定级数\(\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{(n+1)^n}\)的敛散性

解题思路：

先判定绝对收敛性：\(\sum \left|\frac{(-1)^{n+1}}{(n+1)^n}\right| = \sum\frac{1}{(n+1)^n}\)

使用比值判别法：

\(\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = \lim_{n\to\infty}\left[\frac{(n+1)^n}{(n+2)^{n+1}}\right] = \lim_{n\to\infty}\left[\frac{(n+1)}{(n+2)}\right]^n \frac{1}{n+2} = \frac{1}{e} \cdot 0 = 0 < 1\)

所以\(\sum\frac{1}{(n+1)^n}\)收敛，原级数绝对收敛

例3：已知级数\(\sum u_n\)收敛，判断以下级数的敛散性：

(1) \(\sum(-1)^n u_n\)

(2) \(\sum n u_n\)

(3) \(\sum(u_{2n-1} - u_{2n})\)

(4) \(\sum(u_n + u_{n+1})\)

解题思路：

(1) 不能确定。例如：\(u_n = \frac{(-1)^n}{n}\)，\(\sum u_n\)收敛，但\(\sum(-1)^n u_n = \sum\frac{1}{n}\)发散
(2) 不能确定。例如：\(u_n = \frac{(-1)^n}{n}\)，\(\sum u_n\)收敛，但\(\sum n\cdot\frac{(-1)^n}{n} = \sum(-1)^n\)发散
(3) 收敛。因为\(\sum(u_{2n-1} - u_{2n}) = \sum u_{2n-1} - \sum u_{2n}\)，而\(\sum u_{2n-1}\)和\(\sum u_{2n}\)都收敛（因为\(\sum u_n\)收敛）
(4) 收敛。因为\(\sum(u_n + u_{n+1}) = \sum u_n + \sum u_{n+1}\)，而\(\sum u_n\)和\(\sum u_{n+1}\)都收敛
