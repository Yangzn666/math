二、核心概念

1. 空间曲线与曲面方程

空间曲线：

参数方程：x = x(t), y = y(t), z = z(t)
一般方程：F(x,y,z) = 0, G(x,y,z) = 0（两曲面的交线）

空间曲面：

显式方程：z = f(x,y)
一般方程：F(x,y,z) = 0
柱面：二次曲面，如椭圆柱面、双曲柱面、抛物柱面
椭圆柱面：\(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\)
双曲柱面：\(\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\)
抛物柱面：y = ax²
旋转曲面：曲线绕定直线旋转一周形成的曲面

2. 曲线切线与法平面

参数方程曲线：x = x(t), y = y(t), z = z(t)

切线方向向量：\((\frac{dx}{dt}, \frac{dy}{dt}, \frac{dz}{dt})\)
切线方程：\(\frac{x-x_0}{dx/dt} = \frac{y-y_0}{dy/dt} = \frac{z-z_0}{dz/dt}\)
法平面方程：\(\frac{dx}{dt}(x-x_0) + \frac{dy}{dt}(y-y_0) + \frac{dz}{dt}(z-z_0) = 0\)

一般方程曲线：F(x,y,z) = 0, G(x,y,z) = 0

切线方向向量：\((F_yG_z - F_zG_y, F_zG_x - F_xG_z, F_xG_y - F_yG_x)\)
切线方程：\(\frac{x-x_0}{F_yG_z - F_zG_y} = \frac{y-y_0}{F_zG_x - F_xG_z} = \frac{z-z_0}{F_xG_y - F_yG_x}\)
法平面方程：\((F_yG_z - F_zG_y)(x-x_0) + (F_zG_x - F_xG_z)(y-y_0) + (F_xG_y - F_yG_x)(z-z_0) = 0\)

3. 曲面切平面与法线

显式曲面：z = f(x,y)

法向量：\((-f_x, -f_y, 1)\) 或 \((f_x, f_y, -1)\)
切平面方程：\(z - z_0 = f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0)\)
法线方程：\(\frac{x-x_0}{f_x} = \frac{y-y_0}{f_y} = \frac{z-z_0}{-1}\)

一般曲面：F(x,y,z) = 0

法向量：\((F_x, F_y, F_z)\)
切平面方程：\(F_x(x_0,y_0,z_0)(x-x_0) + F_y(x_0,y_0,z_0)(y-y_0) + F_z(x_0,y_0,z_0)(z-z_0) = 0\)
法线方程：\(\frac{x-x_0}{F_x} = \frac{y-y_0}{F_y} = \frac{z-z_0}{F_z}\)

4. 方向导数与梯度

梯度向量：\(\nabla f = (f_x, f_y, f_z)\)
方向导数：\(D_u f = \nabla f \cdot u\)
其中u是单位方向向量，θ是\(\nabla f\)与u的夹角
\(D_u f = \nabla f \cos\theta\)

梯度的性质：

梯度方向是函数值增加最快的方向
梯度大小是函数值增加的最快速率

5. 散度与旋度

散度（div F）：标量

div F = \(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\)
表示向量场在某点的发散程度

旋度（rot F）：向量

rot F = \((\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})\)
表示向量场在某点的旋转程度

三、常见题型及解题思路

1. 求空间曲线的切线与法平面方程

解题思路：

1. 确定曲线的表示形式（参数方程或一般方程）
2. 求出切线方向向量
3. 代入切线方程和法平面方程

例17.1：求曲线x = t, y = t², z = t³在点(1,1,1)处的切线与法平面方程

解题思路：

1. 参数方程：x = t, y = t², z = t³
2. 切线方向向量：\((\frac{dx}{dt}, \frac{dy}{dt}, \frac{dz}{dt}) = (1, 2t, 3t^2)\)
3. 在点(1,1,1)处，t = 1，方向向量为(1, 2, 3)
4. 切线方程：\(\frac{x-1}{1} = \frac{y-1}{2} = \frac{z-1}{3}\)
5. 法平面方程：\(1\cdot(x-1) + 2\cdot(y-1) + 3\cdot(z-1) = 0 \Rightarrow x + 2y + 3z = 6\)

2. 求空间曲面的切平面与法线方程

解题思路：

1. 确定曲面的表示形式（显式或一般方程）
2. 求出法向量
3. 代入切平面方程和法线方程

例17.2：求曲面z = x² + y²在点(1,1,2)处的切平面与法线方程

解题思路：

1. 显式曲面：z = f(x,y) = x² + y²
2. 法向量：\((-f_x, -f_y, 1) = (-2x, -2y, 1)\)
3. 在点(1,1,2)处，法向量为(-2, -2, 1)
4. 切平面方程：\(-2(x-1) - 2(y-1) + 1(z-2) = 0 \Rightarrow -2x - 2y + z = -2\)
5. 法线方程：\(\frac{x-1}{-2} = \frac{y-1}{-2} = \frac{z-2}{1}\)

3. 求方向导数和梯度

解题思路：

1. 求出梯度向量\(\nabla f\)
2. 确定单位方向向量u
3. 计算方向导数\(D_u f = \nabla f \cdot u\)

例17.3：求函数f(x,y,z) = x² + y² + z²在点(1,1,1)处沿向量v = (1,2,3)的方向导数

解题思路：

1. 梯度向量：\(\nabla f = (2x, 2y, 2z)\)
2. 在点(1,1,1)处，\(\nabla f = (2, 2, 2)\)
3. 单位方向向量u = \(\frac{v}{|v|} = \frac{(1,2,3)}{\sqrt{1+4+9}} = \frac{(1,2,3)}{\sqrt{14}}\)
4. 方向导数：\(D_u f = \nabla f \cdot u = (2,2,2)\cdot\frac{(1,2,3)}{\sqrt{14}} = \frac{2+4+6}{\sqrt{14}} = \frac{12}{\sqrt{14}}\)

4. 求散度与旋度

解题思路：

1. 确定向量场F = (P, Q, R)
2. 求出各偏导数
3. 代入散度和旋度公式

例17.4：求向量场F = (x², y², z²)的散度与旋度

解题思路：

1. P = x², Q = y², R = z²
2. 散度：div F = \(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = 2x + 2y + 2z\)
3. 旋度：rot F = \((\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}) = (0-0, 0-0, 0-0) = (0, 0, 0)\)

四、易错点与注意事项

1. 空间曲线的表示

参数方程和一般方程的转换
切线方向向量的求法，特别是一般方程曲线的切线方向向量

2. 曲面的切平面与法线

显式曲面和一般曲面的法向量求法不同
显式曲面z = f(x,y)的法向量为(-fₓ, -fᵧ, 1)或(fₓ, fᵧ, -1)，注意符号

3. 方向导数与梯度

梯度向量是函数的偏导数组成的向量
方向导数需要单位方向向量，不能直接用原向量

4. 散度与旋度

散度是标量，旋度是向量
旋度的计算要注意偏导数的顺序

5. 旋转曲面方程的求法

旋转曲面方程的求法：在母线上任取一点，过该点作纬圆，求出纬圆上任意点的坐标满足的条件

五、重要公式汇总

1. 空间曲线的切线与法平面：
    切线方向向量：\((\frac{dx}{dt}, \frac{dy}{dt}, \frac{dz}{dt})\)
    切线方程：\(\frac{x-x_0}{dx/dt} = \frac{y-y_0}{dy/dt} = \frac{z-z_0}{dz/dt}\)
    法平面方程：\(\frac{dx}{dt}(x-x_0) + \frac{dy}{dt}(y-y_0) + \frac{dz}{dt}(z-z_0) = 0\)
2. 空间曲面的切平面与法线：
    显式曲面：z = f(x,y)
    切平面方程：\(z - z_0 = f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0)\)
    法线方程：\(\frac{x-x_0}{f_x} = \frac{y-y_0}{f_y} = \frac{z-z_0}{-1}\)
    一般曲面：F(x,y,z) = 0
    切平面方程：\(F_x(x_0,y_0,z_0)(x-x_0) + F_y(x_0,y_0,z_0)(y-y_0) + F_z(x_0,y_0,z_0)(z-z_0) = 0\)
    法线方程：\(\frac{x-x_0}{F_x} = \frac{y-y_0}{F_y} = \frac{z-z_0}{F_z}\)
3. 方向导数与梯度：
    梯度向量：\(\nabla f = (f_x, f_y, f_z)\)
    方向导数：\(D_u f = \nabla f \cdot u\)
4. 散度与旋度：
    散度：div F = \(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\)
    旋度：rot F = \((\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})\)

六、典型例题解析

例1：求由过点(0,-1,1)与点(0,0,0)的直线L绕z轴旋转一周所得的旋转曲面位于z≥0的部分，其方程为z²=2x，求该旋转曲面与平面z=0的交线在xOy面上的投影曲线方程

解题思路：

1. 旋转曲面方程：z² = 2x
2. 与平面z=0的交线：z=0, z²=2x ⇒ x=0
3. 交线在xOy面上的投影：x=0, z=0
4. 但题目中要求的是"交线在xOy面上的投影曲线方程"，xOy面上z=0，所以投影曲线方程为x=0

例2：求曲线y² - (z-1)² = 1, x=0绕z轴旋转一周而成的旋转曲面的方程

解题思路：

1. 曲线：y² - (z-1)² = 1, x=0
2. 绕z轴旋转，纬圆上任意点P(x,y,z)满足x² + y² = y₀²
3. 由曲线方程知：y₀² = 1 + (z-1)²
4. 所以旋转曲面方程：x² + y² = 1 + (z-1)² ⇒ x² + y² - (z-1)² = 1

例3：设F(x,y,z) = xy - zlny + eˣ - 1，求在点(0,1,1)处的隐函数存在性

解题思路：

1. 计算偏导数：
    Fₓ = y + eˣ
    Fᵧ = x - z/y
    F_z = -lny + eᶻ
2. 在点(0,1,1)处：
    Fₓ(0,1,1) = 1 + 1 = 2 ≠ 0
    Fᵧ(0,1,1) = 0 - 1/1 = -1 ≠ 0
    F_z(0,1,1) = -ln1 + e¹ = 1 ≠ 0
3. 因此，在点(0,1,1)的某一个邻域内，存在连续且具有连续偏导数的隐函数x=x(y,z)、y=y(x,z)和z=z(x,y)
