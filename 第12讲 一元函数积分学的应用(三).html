二、核心概念

1. 物理应用（仅数学一、数学二）

(1) 变力沿直线做功

基本公式：\(W = \int_a^b F(x) dx\)
物理意义：力\(F(x)\)在位移区间\([a,b]\)上所做的功

(2) 抽水做功

基本公式：\(W = \int_a^b \rho g x A(x) dx\)
其中：
\(\rho\)为水的密度
\(g\)为重力加速度
\(x\)为水深
\(A(x)\)为水深\(x\)处的截面面积

(3) 静水压力

基本公式：\(P = \int_a^b \rho g x [f(x) - h(x)] dx\)
其中：
\(x\)为水深
\(f(x) - h(x)\)为平板在水深\(x\)处的宽度
压力微元\(dP = \rho g x[f(x) - h(x)]dx\)

(4) 引力

基本公式：\(F = Gm\mu \int_a^b \frac{1}{(a-x)^2} dx\)
其中：
\(G\)为引力常量
\(m\)为质点质量
\(\mu\)为细棒的线密度
\(a\)为质点到细棒右端的距离

2. 经济应用（仅数学三）

(1) 边际函数与总量关系

总成本：\(C(Q) = \int_0^Q C'(t) dt + C_0\)
总收益：\(R(Q) = \int_0^Q R'(t) dt\)
总利润：\(L(Q) = R(Q) - C(Q)\)

(2) 需求弹性

定义：\(\eta = - \frac{p}{Q} \cdot \frac{dQ}{dp}\)
经济意义：
\(\eta > 1\)：需求富有弹性，价格下降会增加总收益
\(\eta = 1\)：需求单位弹性，价格变化不影响总收益
\(\eta < 1\)：需求缺乏弹性，价格下降会减少总收益

三、常见题型及解题思路

1. 静水压力问题

解题思路：

1. 建立坐标系，确定水深\(x\)
2. 确定平板在水深\(x\)处的宽度\(f(x) - h(x)\)
3. 写出压力微元\(dP = \rho g x[f(x) - h(x)]dx\)
4. 积分求解总压力

例12.3：斜边长为\(2a\)的等腰直角三角形平板铅直地沉没在水中，且斜边与水面相齐。求该平板一侧所受的水压力。

解：\(P = \int_0^a 2\rho g(a-y)y dy = 2\rho g \left[\frac{ay^2}{2} - \frac{y^3}{3}\right]_0^a = \frac{1}{3}a^3 \rho g\)

2. 变力做功问题

解题思路：

1. 确定变力\(F(x)\)与位移\(x\)的关系
2. 写出功的表达式\(W = \int_a^b F(x) dx\)
3. 计算积分

例：将一质量为\(m\)的物体从地面竖直上抛，初速度为\(v_0\)，求物体上升到最高点所做的功。

解：变力为重力\(F(x) = -mg\)
位移区间：从0到\(h\)，其中\(h = \frac{v_0^2}{2g}\)
\(W = \int_0^h -mg dx = -mgh = -\frac{mv_0^2}{2}\)

3. 经济应用问题

解题思路：

1. 根据题目条件，写出边际成本、边际收益等函数
2. 通过积分求解总成本、总收益等
3. 求解利润最大化问题

例12.4：已知某产品的边际成本为\(4 + \frac{1}{x}\)（万元/单位），固定成本为1万元，产品对价格的需求弹性为\(1 - \frac{p}{8}\)（\(p>0\)），产品最大需求量为8，求使产品取最大利润时的产量和价格。

解：
总成本：\(C(x) = \int_0^x \left(4 + \frac{1}{t}\right) dt + 1 = 4x + \ln x + 1\)
由需求弹性：\(\eta = - \frac{p}{Q} \cdot \frac{dQ}{dp} = 1 - \frac{p}{8}\)
解得：\(x = 8 - p\)，\(R(x) = 8x - x^2\)
利润函数：\(L(x) = R(x) - C(x) = -x^2 + 4x - \ln x - 1\)
令\(L'(x) = -2x + 4 - \frac{1}{x} = 0\)，解得\(x = \frac{16}{9}\)
价格：\(p = 8 - x = \frac{56}{9}\)

四、易错点与注意事项

1. 静水压力问题

压力随水深改变，必须确定水深\(x\)处的平板宽度
注意单位统一，水的密度和重力加速度的取值

2. 变力做功问题

确定变力\(F(x)\)与位移\(x\)的关系
注意功的正负号，与位移方向一致为正

3. 经济应用问题

边际成本是成本函数的导数，不是平均成本
需求弹性与总收益的关系要理解清楚
利润最大化条件是边际收益等于边际成本

4. 积分上下限

确定积分区间时，要注意物理意义
例如静水压力中，水深\(x\)从0到最大水深

5. 物理量的单位

密度\(\rho\)的单位（kg/m³）
重力加速度\(g\)的单位（m/s²）
体积单位（m³）
力的单位（N）

五、重要公式汇总

1. 变力做功：\(W = \int_a^b F(x) dx\)
2. 抽水做功：\(W = \int_a^b \rho g x A(x) dx\)
3. 静水压力：\(P = \int_a^b \rho g x [f(x) - h(x)] dx\)
4. 引力：\(F = Gm\mu \int_a^b \frac{1}{(a-x)^2} dx\)
5. 总成本：\(C(Q) = \int_0^Q C'(t) dt + C_0\)
6. 总收益：\(R(Q) = \int_0^Q R'(t) dt\)
7. 需求弹性：\(\eta = - \frac{p}{Q} \cdot \frac{dQ}{dp}\)

六、典型例题解析

例1：静水压力计算

题目：一矩形平板，高为2m，宽为1m，铅直沉入水中，上边缘与水面相齐，求平板一侧所受的水压力（水的密度为1000kg/m³，重力加速度g=9.8m/s²）。

解题思路：

建立坐标系：水深\(x\)从0到2
平板宽度：平板在水深\(x\)处的宽度为1m
压力微元：\(dP = \rho g x \cdot 1 \cdot dx\)
总压力：\(P = \int_0^2 1000 \cdot 9.8 \cdot x \cdot 1 dx = 9800 \left[\frac{x^2}{2}\right]_0^2 = 9800 \cdot 2 = 19600 \text{ N}\)

例2：经济应用问题

题目：某产品边际成本为\(C'(x) = 2 + x\)（万元/单位），固定成本为5万元，需求函数为\(p = 10 - 0.5x\)，求利润最大时的产量和价格。

解题思路：

总成本：\(C(x) = \int_0^x (2 + t) dt + 5 = 2x + \frac{x^2}{2} + 5\)
总收益：\(R(x) = px = (10 - 0.5x)x = 10x - 0.5x^2\)
利润函数：\(L(x) = R(x) - C(x) = 10x - 0.5x^2 - 2x - \frac{x^2}{2} - 5 = -x^2 + 8x - 5\)
求极值：令\(L'(x) = -2x + 8 = 0\)，得\(x = 4\)
价格：\(p = 10 - 0.5 \cdot 4 = 8\)万元/单位
