二、核心概念

1. 基本求导公式（必须熟记）

函数 导数
\(x^n\) \(nx^{n-1}\)
\(\sin x\) \(\cos x\)
\(\cos x\) \(-\sin x\)
\(e^x\) \(e^x\)
\(\ln x\) \(\frac{1}{x}\)
\(a^x\) \(a^x \ln a\)

2. 导数的四则运算法则

加减法则：

\((u \pm v)' = u' \pm v'\)

乘法法则：

\((uv)' = u'v + uv'\)

除法法则：

\(\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}\)

3. 复合函数求导法则（链式法则）

\((f(g(x)))' = f'(g(x)) \cdot g'(x)\)

一阶微分形式不变性：

\(d[f(u)] = f'(u)\, du\)

4. 高阶导数

二阶导数：

\(f''(x) = (f'(x))'\)

n阶导数：

\(f^{(n)}(x) = (f^{(n-1)}(x))'\)

莱布尼茨公式：

\((uv)^{(n)} = \sum_{k=0}^{n} C_n^k u^{(k)} v^{(n-k)}\)

三、常见题型及解题思路

1. 复合函数求导

解题思路：

1. 确定外层函数和内层函数
2. 先求外层函数对内层函数的导数
3. 再求内层函数对x的导数
4. 两部分相乘

例：求\(y = e^{\sin x}\)的导数

外层函数：\(e^u\)，内层函数：\(u = \sin x\)
外层导数：\(e^u\)
内层导数：\(\cos x\)
结果：\(y' = e^{\sin x} \cdot \cos x\)

2. 分段函数求导

解题思路：

1. 判断分段点处的连续性
2. 对于非分段点，直接求导
3. 对于分段点，分别计算左右导数
4. 比较左右导数是否相等，确定是否可导

例：设\(f(x) = \begin{cases} x^2, & x \geq 1 \\ 2x-1, & x < 1 \end{cases}\)，求\(f'(x)\)

在\(x=1\)处：\(f(1) = 1^2 = 1\)，左极限\(\lim_{x \to 1^-} f(x) = 2 \cdot 1 - 1 = 1\)，右极限\(\lim_{x \to 1^+} f(x) = 1^2 = 1\)，连续
左导数：\(f'_-(1) = \lim_{x \to 1^-} \frac{f(x) - f(1)}{x - 1} = \lim_{x \to 1^-} \frac{2x-1-1}{x-1} = 2\)
右导数：\(f'_+(1) = \lim_{x \to 1^+} \frac{f(x) - f(1)}{x - 1} = \lim_{x \to 1^+} \frac{x^2-1}{x-1} = 2\)
结论：左右导数相等，故\(f(x)\)在\(x=1\)处可导，\(f'(1) = 2\)

3. 隐函数求导

解题思路：

1. 对等式两边同时关于x求导
2. 将y视为x的函数，使用链式法则
3. 解出\(y'\)

例：求\(F(x,y) = x^2 + y^2 - 1 = 0\)的导数

两边对x求导：\(2x + 2y \cdot y' = 0\)
解出\(y'\)：\(y' = -\frac{x}{y}\)

4. 参数方程所确定的函数的导数

解题思路：

1. 求\(\frac{dx}{dt}\)和\(\frac{dy}{dt}\)
2. \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\)

例：\(x = t^2\), \(y = t^3\)，求\(\frac{dy}{dx}\)

计算导数：\(\frac{dx}{dt} = 2t\), \(\frac{dy}{dt} = 3t^2\)
求导结果：\(\frac{dy}{dx} = \frac{3t^2}{2t} = \frac{3t}{2}\)

5. 反函数的导数

解题思路：

1. 若\(y = f(x)\)有反函数\(x = \phi(y)\)，则\(\phi'(y) = \frac{1}{f'(x)}\)
2. 若求反函数的二阶导数，\(\phi''(y) = -\frac{f''(x)}{[f'(x)]^3}\)

例：设\(y = f(x) = 3x + e^x\)有反函数\(x = \phi(y)\)，求\(\phi''(3+e)\)

当\(y = 3 + e\)时：\(x = 1\)
计算导数：\(f'(x) = 3 + e^x\), \(f'(1) = 3 + e\)
计算二阶导数：\(f''(x) = e^x\), \(f''(1) = e\)
反函数二阶导数：\(\phi''(y) = -\frac{f''(x)}{[f'(x)]^3} = -\frac{e}{(3+e)^3}\)
结果：\(\phi''(3+e) = -\frac{e}{(3+e)^3}\)

6. 对数求导法

适用情况：幂指函数或多个函数相乘/相除

解题思路：

1. 取对数：\(\ln y = \ln[f(x)]\)
2. 两边对x求导
3. 解出\(y'\)

例：求\(y = x^x\)的导数

取对数：\(\ln y = x \ln x\)
两边求导：\(\frac{y'}{y} = \ln x + x \cdot \frac{1}{x} = \ln x + 1\)
解出\(y'\)：\(y' = y(\ln x + 1) = x^x(\ln x + 1)\)

7. 高阶导数

解题思路：

1. 先求低阶导数
2. 观察规律，找出通项公式
3. 用数学归纳法证明通项公式

例：求\(f(x) = \sin x\)的n阶导数

计算低阶导数：
\(f'(x) = \cos x = \sin(x + \frac{\pi}{2})\)
\(f''(x) = -\sin x = \sin(x + \pi)\)
\(f'''(x) = -\cos x = \sin(x + \frac{3\pi}{2})\)
\(f^{(4)}(x) = \sin x = \sin(x + 2\pi)\)
通项公式：\(f^{(n)}(x) = \sin(x + \frac{n\pi}{2})\)

四、易错点与注意事项

1. 复合函数求导

必须正确识别内外层函数
不能忽略内层函数的导数

2. 分段函数求导

在分段点处，需要分别计算左右导数
不能直接使用分段点处的表达式求导

3. 隐函数求导

必须将y视为x的函数，使用链式法则
不能将y当作常数处理

4. 参数方程求导

必须计算\(\frac{dx}{dt}\)和\(\frac{dy}{dt}\)
不能直接用\(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\)以外的方法

5. 反函数求导

不能混淆\(f'(x)\)和\(\phi'(y)\)的关系
二阶导数的公式需要特别注意

6. 对数求导法

适用于幂指函数或多个函数相乘/相除
不能用于简单函数的求导

7. 高阶导数

高阶导数的计算需要耐心和观察力
莱布尼茨公式适用于两个函数乘积的高阶导数

五、重要公式汇总

1. 复合函数求导：\((f(g(x)))' = f'(g(x)) \cdot g'(x)\)
2. 一阶微分形式不变性：\(d[f(u)] = f'(u) \, du\)
3. 隐函数求导：\(F(x,y) = 0 \Rightarrow \frac{dy}{dx} = -\frac{F_x}{F_y}\)
4. 参数方程求导：\(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\)
5. 反函数求导：\(\phi'(y) = \frac{1}{f'(x)}\)
6. 反函数二阶导数：\(\phi''(y) = -\frac{f''(x)}{[f'(x)]^3}\)
7. 莱布尼茨公式：\((uv)^{(n)} = \sum_{k=0}^{n} C_n^k u^{(k)} v^{(n-k)}\)
