二、核心概念

1. 平面图形的面积

直角坐标系下：

由\(y = f(x), y = g(x), x = a, x = b\)围成的图形面积：
\(S = \int_a^b |f(x) - g(x)| dx\)
由\(x = f(y), x = g(y), y = c, y = d\)围成的图形面积：
\(S = \int_c^d |f(y) - g(y)| dy\)

极坐标系下：

由\(r = f(\theta), \theta = \alpha, \theta = \beta\)围成的图形面积：
\(S = \frac{1}{2}\int_\alpha^\beta r^2 d\theta\)

2. 旋转体的体积

绕x轴旋转：

\(y = f(x), a \leq x \leq b, f(x) \geq 0\)：
\(V = \pi\int_a^b [f(x)]^2 dx\)

绕y轴旋转：

\(y = f(x), a \leq x \leq b, f(x) \geq 0\)：
\(V = 2\pi\int_a^b x f(x) dx\)

绕任意直线旋转：

平面曲线\(L\)：\(y = f(x), a \leq x \leq b\)，且\(f(x)\)可导
定直线\(L_0\)：\(Ax + By + C = 0\)
旋转体体积：
\(V = \pi\int_a^b \frac{(Ax + Bf(x) + C)^2}{A^2 + B^2} dx\)
特别地，当\(L_0\)为x轴(\(y=0\))：
\(V = \pi\int_a^b [f(x)]^2 dx\)

3. 函数的平均值

函数\(f(x)\)在\([a,b]\)上的平均值：
\(f_{avg} = \frac{1}{b-a}\int_a^b f(x) dx\)

4. 形心坐标公式

形心坐标\((\bar{x}, \bar{y})\)：
\(\bar{x} = \frac{1}{S}\int_a^b x|f(x) - g(x)| dx\)
\(\bar{y} = \frac{1}{S}\int_a^b \frac{f(x) + g(x)}{2}|f(x) - g(x)| dx\)

5. 平面曲线的弧长

\(y = f(x), a \leq x \leq b\)：
\(L = \int_a^b \sqrt{1 + (f'(x))^2} dx\)

6. 旋转曲面的面积(侧面积)

\(y = f(x), a \leq x \leq b, f(x) \geq 0\)，绕x轴旋转：
\(S = 2\pi\int_a^b f(x) \sqrt{1 + (f'(x))^2} dx\)

三、常见题型及解题思路

1. 平面图形的面积计算

解题思路：

1. 确定图形边界：确定上下(左右)边界函数
2. 确定积分区间：找到交点确定a和b
3. 写出面积公式：\(S = \int_a^b |\text{上边界} - \text{下边界}| dx\)
4. 计算积分

例：求\(y = x^2\)与\(y = x\)所围平面图形的面积

交点：\(x^2 = x \Rightarrow x = 0, x = 1\)
边界：上边界：\(y = x\)，下边界：\(y = x^2\)
面积：\(S = \int_0^1 (x - x^2) dx = \left[\frac{x^2}{2} - \frac{x^3}{3}\right]_0^1 = \frac{1}{6}\)

2. 旋转体的体积计算

解题思路：

1. 确定旋转轴
2. 确定积分区间
3. 选择适当的体积公式
4. 计算积分

例：求\(y = x^2\)，\(x = 0\)，\(x = 1\)，\(y = 0\)所围图形绕x轴旋转的体积

旋转轴：x轴
积分区间：\([0,1]\)
体积公式：\(V = \pi\int_0^1 (x^2)^2 dx = \pi\int_0^1 x^4 dx = \frac{\pi}{5}\)

3. 利用反函数求旋转体体积

解题思路：

1. 将\(y = f(x)\)反解为\(x = \varphi(y)\)
2. 确定旋转轴和积分区间
3. 选择适当的体积公式
4. 计算积分

例：求\(y = \sqrt{1 + x^2}\)，\(x \geq 0\)，\(y = 0\)，\(y = 2\)所围图形绕x轴旋转的体积

反解：\(x = \sqrt{y^2 - 1}\)
旋转轴：x轴
积分区间：\([1,2]\)
体积公式：\(V = \pi\int_1^2 x^2 dy = \pi\int_1^2 (y^2 - 1) dy = \pi\left[\frac{y^3}{3} - y\right]_1^2 = \frac{4\pi}{3}\)

4. 绕任意直线旋转的体积计算

解题思路：

1. 确定直线方程\(Ax + By + C = 0\)
2. 确定积分区间
3. 应用公式\(V = \pi\int_a^b \frac{(Ax + Bf(x) + C)^2}{A^2 + B^2} dx\)
4. 计算积分

例：求\(y = x\)，\(x = 0\)，\(x = 1\)所围图形绕直线\(y = x + 1\)旋转的体积

直线方程：\(x - y + 1 = 0\)，\(A = 1\)，\(B = -1\)，\(C = 1\)
体积公式：\(V = \pi\int_0^1 \frac{(x - x + 1)^2}{1^2 + (-1)^2} dx = \pi\int_0^1 \frac{1}{2} dx = \frac{\pi}{2}\)

四、易错点与注意事项

1. 旋转轴的确定

确定旋转轴是绕x轴、y轴还是其他直线
不同旋转轴对应不同的体积公式

2. 积分区间的确定

找到图形的交点，确定积分上下限
注意图形在不同区间可能有不同的边界

3. 绝对值的处理

在计算面积时，需要考虑上下(左右)边界函数的大小关系
通常使用绝对值来处理，避免负面积

4. 反函数的求解

有时需要将\(y = f(x)\)反解为\(x = \varphi(y)\)
确保反解正确，注意定义域和值域

5. 绕任意直线旋转的公式应用

不能直接套用绕坐标轴的公式
必须使用绕任意直线的通用公式

6. 特殊曲线的处理

伯努利双纽线等特殊曲线需要利用对称性简化计算
振荡曲线需要处理绝对值，考虑振荡部分

六、典型例题解析

例1：求\(y = e^{-x} \sin x(x \geq 0)\)与x轴所围平面图形的面积

解题思路：

确定交点：\(e^{-x} \sin x = 0 \Rightarrow x = k\pi, k = 0,1,2,...\)
确定积分区间：\([k\pi, (k+1)\pi]\)
写出面积公式：\(S = \sum_{k=0}^{\infty} \int_{k\pi}^{(k+1)\pi} e^{-x} \sin x dx\)
处理绝对值：由于\(\sin x\)在\([k\pi, (k+1)\pi]\)上符号交替，\(|\sin x| = \sin x\)当k为偶数，\(|\sin x| = -\sin x\)当k为奇数
计算：\(S = \sum_{k=0}^{\infty} (-1)^k \int_{k\pi}^{(k+1)\pi} e^{-x} \sin x dx\)
\(= \sum_{k=0}^{\infty} (-1)^k \left[e^{-x} \frac{-\cos x - \sin x}{2}\right]_{k\pi}^{(k+1)\pi}\)
\(= \frac{1}{2}\sum_{k=0}^{\infty} (-1)^k e^{-k\pi} [(-1)^{k+1} - (-1)^k]\)
\(= \frac{1}{2}\sum_{k=0}^{\infty} e^{-k\pi}\)
\(= \frac{1}{2} \cdot \frac{1}{1 - e^{-\pi}}\)

例2：求伯努利双纽线\(r^2 = a^2 \cos 2\theta\)围成的图形的面积

解题思路：

画图：利用对称性，图形在第一象限和第四象限对称
确定积分区间：\(\theta \in [-\frac{\pi}{4}, \frac{\pi}{4}]\)
写出面积公式：\(S = 4 \cdot \frac{1}{2}\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} r^2 d\theta = 2\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} a^2 \cos 2\theta d\theta\)
计算：\(S = 2a^2 \int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} \cos 2\theta d\theta = 2a^2 \left[\frac{\sin 2\theta}{2}\right]_{-\frac{\pi}{4}}^{\frac{\pi}{4}} = a^2 (\sin(\frac{\pi}{2}) - \sin(-\frac{\pi}{2})) = a^2\)
