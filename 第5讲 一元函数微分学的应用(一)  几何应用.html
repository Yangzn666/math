二、核心概念

1. 极值的定义

极大值点：若存在\(x_0\)的某个邻域，使得在该邻域内任意一点\(x\)，均有\(f(x) \leq f(x_0)\)成立，则称\(x_0\)为\(f(x)\)的极大值点，\(f(x_0)\)为极大值。

极小值点：若存在\(x_0\)的某个邻域，使得在该邻域内任意一点\(x\)，均有\(f(x) \geq f(x_0)\)成立，则称\(x_0\)为\(f(x)\)的极小值点，\(f(x_0)\)为极小值。

2. 极值的必要条件

若\(f(x)\)在\(x_0\)处可导，且\(x_0\)为极值点，则\(f'(x_0) = 0\)。

3. 极值的充分条件

一阶导数判别法：若\(f'(x)\)在\(x_0\)的左右邻域内变号（由正变负或由负变正），则\(x_0\)为极值点。

由正变负：\(x_0\)为极大值点
由负变正：\(x_0\)为极小值点

二阶导数判别法：若\(f'(x_0) = 0\)，且\(f''(x_0) \neq 0\)，则

\(f''(x_0) < 0\)：\(x_0\)为极大值点
\(f''(x_0) > 0\)：\(x_0\)为极小值点

4. 函数在闭区间\([a,b]\)上的最值

最大值和最小值必存在
最大值和最小值可能在区间内部的极值点或在区间端点处取得

5. 拐点的定义

拐点：曲线在某点处凹凸性发生变化的点。

6. 拐点的必要条件

设\(f''(x_0)\)存在，且点\((x_0, f(x_0))\)为曲线的拐点，则\(f''(x_0) = 0\)。

注意：\(f''(x_0)\)不存在的点也有可能是拐点。

7. 拐点的充分条件

设\(f(x)\)在点\(x=x_0\)处连续，在点\(x=x_0\)的某去心邻域\(U(x_0, \delta)\)内二阶导数存在，且在该点的左、右邻域内\(f''(x)\)变号（由正变负或由负变正），则点\((x_0, f(x_0))\)为曲线的拐点。

三、常见题型及解题思路

1. 求函数的极值

解题思路：

1. 求函数的导数\(f'(x)\)
2. 找出\(f'(x) = 0\)的点和\(f'(x)\)不存在的点（临界点）
3. 判断临界点是否为极值点
    一阶导数判别法：判断\(f'(x)\)在临界点左右邻域的符号变化
    二阶导数判别法：计算\(f''(x)\)在临界点的值
4. 确定极值点和极值

例：求\(f(x) = x^3 - 3x\)的极值

求导：\(f'(x) = 3x^2 - 3 = 3(x^2 - 1)\)
找临界点：令\(f'(x) = 0\)，得\(x = -1, x = 1\)
二阶导数：\(f''(x) = 6x\)
判断：\(f''(-1) = -6 < 0\)，故\(x = -1\)为极大值点，极大值为\(f(-1) = 2\)
判断：\(f''(1) = 6 > 0\)，故\(x = 1\)为极小值点，极小值为\(f(1) = -2\)

2. 求函数在闭区间上的最值

解题思路：

1. 求函数在区间内部的临界点
2. 计算临界点处的函数值
3. 计算区间端点处的函数值
4. 比较所有函数值，最大者为最大值，最小者为最小值

例：求\(f(x) = x^3 - 3x\)在\([-2, 2]\)上的最大值和最小值

临界点：\(x = -1, x = 1\)（同上例）
计算函数值：
\(f(-2) = -8 + 6 = -2\)
\(f(-1) = 2\)（极大值）
\(f(1) = -2\)（极小值）
\(f(2) = 8 - 6 = 2\)
比较：最大值为2（在\(x = -1\)和\(x = 2\)处取得），最小值为-2（在\(x = -2\)和\(x = 1\)处取得）

3. 判断拐点

解题思路：

1. 求函数的二阶导数\(f''(x)\)
2. 找出\(f''(x) = 0\)的点和\(f''(x)\)不存在的点
3. 判断这些点是否为拐点
    判断\(f''(x)\)在这些点左右邻域的符号变化
4. 确定拐点

例：判断\(f(x) = x^3\)的拐点

求导：\(f'(x) = 3x^2\)
二阶导数：\(f''(x) = 6x\)
找临界点：令\(f''(x) = 0\)，得\(x = 0\)
判断符号：当\(x < 0\)时，\(f''(x) < 0\)（下凸）；当\(x > 0\)时，\(f''(x) > 0\)（上凸）
结论：\(f''(x)\)在\(x = 0\)处由负变正，故点\((0, 0)\)为拐点

四、易错点与注意事项

1. 极值点与驻点的关系

驻点：\(f'(x_0) = 0\)的点
极值点一定是驻点或不可导点，但驻点不一定是极值点

2. 拐点与二阶导数为0的关系

拐点处二阶导数不一定为0（可能不存在）
二阶导数为0的点不一定是拐点

3. 极值与最值的区别

极值是局部性质，最值是全局性质
函数在区间上可能有多个极值，但只有一个最大值和一个最小值

4. 闭区间上最值的求法

闭区间上的连续函数必有最大值和最小值
最大值和最小值可能在区间内部的极值点或在区间端点处取得

5. 端点处的极值

端点处不能用一阶导数判别法（因为没有左右邻域）
端点处的极值需要通过比较函数值来确定

6. 二阶导数不存在的点

二阶导数不存在的点也可能是拐点
例如：\(f(x) = x^{1/3}\)在\(x = 0\)处

五、重要结论

1. 极值的必要条件：\(f'(x_0) = 0\)（若\(f(x)\)在\(x_0\)处可导）
2. 极值的充分条件：
    一阶导数变号：由正变负为极大值，由负变正为极小值
    二阶导数不为0：\(f''(x_0) < 0\)为极大值，\(f''(x_0) > 0\)为极小值
3. 拐点的必要条件：\(f''(x_0) = 0\)（若\(f''(x_0)\)存在）
4. 拐点的充分条件：\(f''(x)\)在\(x_0\)的左右邻域内变号
5. 闭区间上连续函数的最值：必存在，且在区间内部的极值点或端点处取得
6. 拐点的定义：曲线在某点处凹凸性发生变化的点
7. 极值点是局部概念，拐点是曲线凹凸性变化的点
