二、核心概念

1. 分部积分法

基本公式：\(\int u \, dv = uv - \int v \, du\)

适用情况：被积函数为两个函数的乘积，且其中一个函数容易求导，另一个函数容易积分

2. 分部积分法的推广公式

公式：\(\int u v^{(n+1)} dx = uv^{(n)} - u'v^{(n-1)} + u''v^{(n-2)} - ... + (-1)^n u^{(n)}v + (-1)^{n+1}\int u^{(n+1)}v \, dx\)

表格法：将\(u\)的各阶导数和\(v\)的各阶原函数按表格排列，左上右下错位相乘，符号交替

3. 特殊类型的积分

\(e^x \sin bx\)型：\(\int e^x \sin bx \, dx = \frac{e^x (a \sin bx - b \cos bx)}{a^2 + b^2} + C\)
\(e^x \cos bx\)型：\(\int e^x \cos bx \, dx = \frac{e^x (a \cos bx + b \sin bx)}{a^2 + b^2} + C\)

三、常见题型及解题思路

1. 分部积分法的使用

解题思路：

1. 识别被积函数是否为两个函数的乘积
2. 选择\(u\)和\(dv\)，使\(u\)容易求导，\(v\)容易积分
3. 应用分部积分公式：\(\int u \, dv = uv - \int v \, du\)
4. 重复使用分部积分法，直到得到可计算的积分

例：计算\(\int x^2 e^x dx\)

选择：\(u = x^2, dv = e^x dx\)
计算：\(du = 2x dx, v = e^x\)
应用公式：\(\int x^2 e^x dx = x^2 e^x - \int 2x e^x dx\)
再次应用：\(\int 2x e^x dx = 2x e^x - 2\int e^x dx = 2x e^x - 2e^x + C\)
最终结果：\(\int x^2 e^x dx = x^2 e^x - 2x e^x + 2e^x + C\)

2. 特殊类型的积分计算

解题思路：

1. 识别积分类型（\(e^x \sin bx\)或\(e^x \cos bx\)）
2. 应用特殊公式直接计算
3. 或者通过两次分部积分法推导

例：计算\(\int e^{2x} \sin 3x dx\)

设：\(I = \int e^{2x} \sin 3x dx\)
第一次分部积分：\(u = \sin 3x, dv = e^{2x} dx\)
\(du = 3\cos 3x dx, v = \frac{1}{2}e^{2x}\)
\(I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{2}\int e^{2x} \cos 3x dx\)
第二次分部积分：\(\int e^{2x} \cos 3x dx\)
\(u = \cos 3x, dv = e^{2x} dx\)
\(du = -3\sin 3x dx, v = \frac{1}{2}e^{2x}\)
\(\int e^{2x} \cos 3x dx = \frac{1}{2}e^{2x} \cos 3x + \frac{3}{2}\int e^{2x} \sin 3x dx\)
代入：\(I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{2}\left[\frac{1}{2}e^{2x} \cos 3x + \frac{3}{2}I\right]\)
\(I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{4}e^{2x} \cos 3x - \frac{9}{4}I\)
解得：\(\frac{13}{4}I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{4}e^{2x} \cos 3x\)
\(I = \frac{2}{13}e^{2x} \sin 3x - \frac{3}{13}e^{2x} \cos 3x + C\)

3. 通过变量替换求解\(\int f(\ln x)dx\)

解题思路：

1. 令\(t = \ln x\)，将\(x = e^t\)代入
2. 将积分转化为\(\int f(t) e^t dt\)
3. 应用分部积分法或其他方法求解

例：设\(f(\ln x) = \ln(1+x)\)，计算\(\int \frac{f(x)}{x}dx\)

变量替换：令\(t = \ln x\)，则\(x = e^t\)，\(f(t) = \ln(1+e^t)\)
转换积分：\(\int \frac{f(x)}{x}dx = \int \ln(1+e^t) \frac{e^t}{e^t} dt = \int \ln(1+e^t) dt\)
分部积分：令\(u = \ln(1+e^t), dv = dt\)
\(du = \frac{e^t}{1+e^t} dt, v = t\)
\(\int \ln(1+e^t) dt = t \ln(1+e^t) - \int t \frac{e^t}{1+e^t} dt\)
化简：\(\int t \frac{e^t}{1+e^t} dt = \int t \left[1 - \frac{1}{1+e^t}\right] dt = \int t dt - \int \frac{t}{1+e^t} dt\)
最终结果：\(\int \frac{f(x)}{x}dx = x \ln(1+e^x) - x + \ln(1+e^x) + C\)

4. 三角函数的平方展开

解题思路：

1. 展开平方，利用三角恒等式化简
2. 应用基本积分公式求解

例：计算\(\int e^{2x}(\tan x + 1)^2 dx\)

展开平方：\((\tan x + 1)^2 = \tan^2 x + 2\tan x + 1 = \sec^2 x + 2\tan x\)
原式：\(\int e^{2x}(\sec^2 x + 2\tan x) dx = \int e^{2x} \sec^2 x dx + 2\int e^{2x} \tan x dx\)
求解：利用分部积分法或观察法求解

四、易错点与注意事项

1. 分部积分法的选择

选择\(u\)和\(dv\)时，应使\(u\)容易求导，\(v\)容易积分
通常遵循"反对幂指三"的顺序选择\(u\)

2. 特殊类型的积分公式

记住\(e^x \sin bx\)和\(e^x \cos bx\)的积分公式
不能直接套用，需要根据具体系数调整

3. 变量替换的处理

在替换\(t = \ln x\)时，要正确地将\(dx\)转换为\(dt\)
注意\(x = e^t\)，\(dx = e^t dt\)

4. 三角函数的恒等变换

利用三角恒等式化简被积函数
如\(\tan^2 x = \sec^2 x - 1\)

5. 分部积分法的重复使用

有时需要多次应用分部积分法
每次应用后，要检查是否简化了问题

五、重要公式汇总

1. 分部积分法基本公式：
    \(\int u \, dv = uv - \int v \, du\)
2. 特殊积分公式：
    \(\int e^x \sin bx \, dx = \frac{e^x (a \sin bx - b \cos bx)}{a^2 + b^2} + C\)
    \(\int e^x \cos bx \, dx = \frac{e^x (a \cos bx + b \sin bx)}{a^2 + b^2} + C\)
3. 变量替换法：
    令\(t = \ln x\)，则\(x = e^t\)，\(dx = e^t dt\)
4. 三角恒等式：
    \(\tan^2 x = \sec^2 x - 1\)
    \(\sec^2 x = 1 + \tan^2 x\)

六、典型例题解析

例1：计算\(\int x^3 e^x dx\)

解题思路：

第一步：令\(u = x^3, dv = e^x dx\)
\(du = 3x^2 dx, v = e^x\)
第二步：\(\int x^3 e^x dx = x^3 e^x - 3\int x^2 e^x dx\)
第三步：对\(\int x^2 e^x dx\)重复分部积分：
\(= x^2 e^x - 2\int x e^x dx\)
第四步：对\(\int x e^x dx\)重复分部积分：
\(= x e^x - \int e^x dx = x e^x - e^x + C\)
第五步：代入回原式：
\(\int x^3 e^x dx = x^3 e^x - 3[x^2 e^x - 2(x e^x - e^x)] + C\)
\(= x^3 e^x - 3x^2 e^x + 6x e^x - 6e^x + C\)

例2：计算\(\int e^{2x} \sin 3x dx\)

解题思路：

第一步：令\(I = \int e^{2x} \sin 3x dx\)
第二步：第一次分部积分：
\(u = \sin 3x, dv = e^{2x} dx\)
\(du = 3\cos 3x dx, v = \frac{1}{2}e^{2x}\)
\(I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{2}\int e^{2x} \cos 3x dx\)
第三步：第二次分部积分：
\(\int e^{2x} \cos 3x dx = \frac{1}{2}e^{2x} \cos 3x + \frac{3}{2}\int e^{2x} \sin 3x dx = \frac{1}{2}e^{2x} \cos 3x + \frac{3}{2}I\)
第四步：代入\(I\)的表达式：
\(I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{2}\left[\frac{1}{2}e^{2x} \cos 3x + \frac{3}{2}I\right]\)
\(I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{4}e^{2x} \cos 3x - \frac{9}{4}I\)
第五步：\(\frac{13}{4}I = \frac{1}{2}e^{2x} \sin 3x - \frac{3}{4}e^{2x} \cos 3x\)
\(I = \frac{2}{13}e^{2x} \sin 3x - \frac{3}{13}e^{2x} \cos 3x + C\)
