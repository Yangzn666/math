二、核心概念

1. 与一元函数的区别

一元函数：可微 \(\Rightarrow\) 可导
二元函数：可微 \(\Rightarrow\) 偏导数存在

2. 可微的定义

若\(\Delta z = f(x+\Delta x, y+\Delta y) - f(x, y) = A\Delta x + B\Delta y + o(\rho)\)，其中\(\rho = \sqrt{\Delta x^2 + \Delta y^2}\)，则称函数\(z = f(x, y)\)在点\((x, y)\)处可微
记\(dz = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy\)

3. 可微的充分条件

若函数\(z = f(x, y)\)在点\((x, y)\)处的偏导数存在且连续，则该函数在点\((x, y)\)处可微

4. 隐函数存在定理

若\(F(x, y)\)在点\((x_0, y_0)\)处连续，且\(F_y(x_0, y_0) \neq 0\)，则存在\((x_0, y_0)\)的邻域，使得在该邻域内\(F(x, y) = 0\)确定了\(y = f(x)\)的隐函数，且\(f'(x) = -\frac{F_x}{F_y}\)

5. 二元函数的极值

无条件极值：函数在定义域内部的极值
条件极值：函数在约束条件下的极值

三、常见题型及解题思路

1. 求全微分

解题思路：

1. 计算偏导数\(\frac{\partial z}{\partial x}\)和\(\frac{\partial z}{\partial y}\)
2. 写出全微分\(dz = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy\)

例：求\(z = x^2 + y^2\)的全微分

\(\frac{\partial z}{\partial x} = 2x, \frac{\partial z}{\partial y} = 2y\)
\(dz = 2x dx + 2y dy\)

2. 判断可微性

解题思路：

1. 检查偏导数是否存在
2. 检查偏导数是否连续
3. 若偏导数存在且连续，则函数可微

例：判断\(f(x, y) = \sqrt{x^2 + y^2}\)在\((0,0)\)处是否可微

\(\frac{\partial f}{\partial x} = \frac{x}{\sqrt{x^2 + y^2}}, \frac{\partial f}{\partial y} = \frac{y}{\sqrt{x^2 + y^2}}\)
在\((0,0)\)处偏导数不存在，故不可微

3. 求隐函数的偏导数

解题思路：

1. 对等式两边关于\(x\)或\(y\)求偏导
2. 将\(y\)视为\(x\)的函数，使用链式法则
3. 解出偏导数

例：求\(F(x, y) = x^2 + y^2 - 1 = 0\)的偏导数

两边对\(x\)求偏导：\(2x + 2y\frac{\partial y}{\partial x} = 0\)
\(\frac{\partial y}{\partial x} = -\frac{x}{y}\)

4. 求二元函数的极值

解题思路：

1. 求一阶偏导数，找驻点
2. 求二阶偏导数
3. 判断驻点是否为极值点
4. 计算极值

例：求\(f(x, y) = x^2 + y^2\)的极值

\(\frac{\partial f}{\partial x} = 2x, \frac{\partial f}{\partial y} = 2y\)
驻点：\((0, 0)\)
二阶偏导数：\(f_{xx} = 2, f_{xy} = 0, f_{yy} = 2\)
\(AC - B^2 = 2 \cdot 2 - 0^2 = 4 > 0, f_{xx} > 0\)
\((0, 0)\)为极小值点，极小值为0

5. 求条件极值（拉格朗日乘数法）

解题思路：

1. 构造拉格朗日函数\(L(x, y, \lambda) = f(x, y) + \lambda g(x, y)\)
2. 求\(L\)对\(x, y, \lambda\)的偏导数，令其为0
3. 解方程组，得到驻点
4. 判断驻点是否为极值点

例：求\(f(x, y) = x^2 + y^2\)在约束条件\(x + y = 1\)下的极值

构造\(L(x, y, \lambda) = x^2 + y^2 + \lambda(x + y - 1)\)
\(\frac{\partial L}{\partial x} = 2x + \lambda = 0\)
\(\frac{\partial L}{\partial y} = 2y + \lambda = 0\)
\(\frac{\partial L}{\partial \lambda} = x + y - 1 = 0\)
解得：\(x = y = \frac{1}{2}, \lambda = -1\)
极小值：\(f(\frac{1}{2}, \frac{1}{2}) = \frac{1}{2}\)

四、易错点与注意事项

1. 可微与偏导数存在

二元函数偏导数存在不一定可微
可微一定偏导数存在

2. 极值点与驻点

极值点一定是驻点或不可导点
驻点不一定是极值点

3. 条件极值的求解

拉格朗日乘数法适用于约束条件为等式的情况
需要检查边界点

4. 隐函数求导

必须将\(y\)视为\(x\)的函数，使用链式法则
不能将\(y\)当作常数处理

5. 二元函数极值的判定

二阶导数判别法：\(AC - B^2 > 0\)时，\(A > 0\)为极小值，\(A < 0\)为极大值
\(AC - B^2 < 0\)时，不是极值点

五、重要公式汇总

1. 全微分：\(dz = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy\)
2. 隐函数求导：
    \(\frac{dy}{dx} = -\frac{F_x}{F_y}\)
    \(\frac{dx}{dy} = -\frac{F_y}{F_x}\)
3. 二元函数极值判定：
    设\(A = f_{xx}, B = f_{xy}, C = f_{yy}\)
    若\(AC - B^2 > 0\)：
        \(A > 0\)：极小值
        \(A < 0\)：极大值
    若\(AC - B^2 < 0\)：不是极值点
    若\(AC - B^2 = 0\)：无法判定
4. 拉格朗日乘数法：
    \(L(x, y, \lambda) = f(x, y) + \lambda g(x, y)\)
    \(\frac{\partial L}{\partial x} = 0, \frac{\partial L}{\partial y} = 0, \frac{\partial L}{\partial \lambda} = 0\)

六、典型例题解析

例1：求曲线\(y = x^2 + ax\)与直线\(y = bx\)所围区域面积的最大值和最小值（\(a \leq 0\)，\(b \geq 0\)，\(a^2 + b^2 = 1\)）

解题思路：

求交点：\(x = 0, x = b - a\)
面积：\(S = \int_0^{b-a} (bx - x^2 - ax) dx = \frac{(b - a)^3}{6}\)
转化：转化为求\((b - a)^3\)的最值，即求\(b - a\)的最值
拉格朗日乘数法：
\(F(a, b, \lambda) = b - a + \lambda(a^2 + b^2 - 1)\)
\(F_a = -1 + 2\lambda a = 0, F_b = 1 + 2\lambda b = 0, F_\lambda = a^2 + b^2 - 1 = 0\)
解得：\(a = -\frac{1}{\sqrt{2}}, b = \frac{1}{\sqrt{2}}\)
最大值：\(S = \frac{(1/\sqrt{2} + 1/\sqrt{2})^3}{6} = \frac{\sqrt{2}}{3}\)
边界点：\(a = 0, b = 1\)时\(S = \frac{1}{6}\)；\(a = -1, b = 0\)时\(S = \frac{1}{6}\)
比较：最大值为\(\frac{\sqrt{2}}{3}\)，最小值为\(\frac{1}{6}\)

例2：求函数\(f(x, y) = x^2 + 2y^2\)在约束条件\(x + y = 1\)下的极值

解题思路：

构造拉格朗日函数：\(L(x, y, \lambda) = x^2 + 2y^2 + \lambda(x + y - 1)\)
求偏导数：
\(\frac{\partial L}{\partial x} = 2x + \lambda = 0\)
\(\frac{\partial L}{\partial y} = 4y + \lambda = 0\)
\(\frac{\partial L}{\partial \lambda} = x + y - 1 = 0\)
解方程组：
\(2x + \lambda = 0\)
\(4y + \lambda = 0\)
\(x + y = 1\)
解得：\(x = \frac{2}{3}, y = \frac{1}{3}, \lambda = -\frac{4}{3}\)
极小值：\(f(\frac{2}{3}, \frac{1}{3}) = (\frac{2}{3})^2 + 2(\frac{1}{3})^2 = \frac{6}{9} = \frac{2}{3}\)
