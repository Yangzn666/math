二、核心概念

1. 一阶微分方程

(1) 可分离变量的微分方程

形式：\(\frac{dy}{dx} = f(x)g(y)\)

解法：分离变量，\(\int\frac{dy}{g(y)} = \int f(x)dx\)

例：\(\frac{dy}{dx} = \frac{x+1}{y-1} \Rightarrow \int(y-1)dy = \int(x+1)dx\)

(2) 一阶线性微分方程

形式：\(y' + p(x)y = q(x)\)

通解：\(y = e^{-\int p(x)dx}[\int q(x)e^{\int p(x)dx}dx + C]\)

解法：求积分因子\(\mu(x) = e^{\int p(x)dx}\)，然后求解

(3) 伯努利方程

形式：\(y' + p(x)y = q(x)y^n (n \neq 0, 1)\)

解法：

1. 令\(z = y^{1-n}\)，则\(y' = \frac{(1-n)z'}{y^n}\)
2. 方程转化为一阶线性微分方程：\(z' + (1-n)p(x)z = (1-n)q(x)\)

2. 二阶常系数微分方程

(1) 二阶常系数齐次线性微分方程

形式：\(y'' + py' + qy = 0\)

特征方程：\(r^2 + pr + q = 0\)

通解：

两个不等实根\(r_1, r_2\)：\(y = C_1e^{r_1x} + C_2e^{r_2x}\)
两个相等实根\(r_1 = r_2\)：\(y = (C_1 + C_2x)e^{r_1x}\)
一对共轭复根\(r = \alpha \pm \beta i\)：\(y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)\)

(2) 二阶常系数非齐次线性微分方程

形式：\(y'' + py' + qy = f(x)\)

通解：齐次方程通解 + 非齐次方程特解

特解求法：待定系数法

若\(f(x) = e^{\alpha x}P_m(x)\)，其中\(P_m(x)\)为\(m\)次多项式，则特解设为\(x^k e^{\alpha x}Q_m(x)\)

\(k = 0, 1, 2\)取决于\(\alpha\)是否为特征根（\(k=0\)表示\(\alpha\)不是特征根，\(k=1\)表示\(\alpha\)是单根，\(k=2\)表示\(\alpha\)是重根）

3. 欧拉方程

形式：\(x^2y'' + axy' + by = 0 (x > 0)\)

解法：令\(x = e^t\)，\(t = \ln x\)

代入后，方程转化为常系数线性微分方程

解出\(y\)关于\(t\)的函数，再代回\(x = e^t\)

三、常见题型及解题思路

1. 一阶线性微分方程的求解

解题思路：

1. 确定方程是否为一阶线性微分方程
2. 求积分因子\(\mu(x) = e^{\int p(x)dx}\)
3. 通解：\(y = \frac{1}{\mu(x)}[\int\mu(x)q(x)dx + C]\)

例15.1：解\(y' - 2y = e^x\)

\(p(x) = -2, q(x) = e^x\)

\(\mu(x) = e^{\int-2dx} = e^{-2x}\)

通解：\(y = e^{2x}[\int e^{-2x}\cdot e^x dx + C] = e^{2x}[\int e^{-x}dx + C] = e^{2x}[-e^{-x} + C] = Ce^{2x} - e^x\)

2. 伯努利方程的求解

解题思路：

1. 判断是否为伯努利方程
2. 令\(z = y^{1-n}\)，将方程转化为一阶线性微分方程
3. 求解一阶线性微分方程
4. 代回\(z = y^{1-n}\)，得到原方程的通解

例15.2：解\(y' + y = xy^2\)

\(n = 2\)，令\(z = y^{1-2} = y^{-1}\)

\(y = z^{-1}, y' = -z^{-2}z'\)

代入方程：\(-z^{-2}z' + z^{-1} = xz^{-2}\)

乘以\(-z^2\)：\(z' - z = -x\)

积分因子\(\mu(x) = e^{-\int dx} = e^{-x}\)

通解：\(z = e^x[\int-xe^{-x}dx + C] = e^x[xe^{-x} + e^{-x} + C] = x + 1 + Ce^x\)

代回\(y = \frac{1}{z}\)：\(y = \frac{1}{x + 1 + Ce^x}\)

3. 二阶常系数非齐次线性微分方程的求解

解题思路：

1. 求齐次方程的通解
2. 求非齐次方程的特解
3. 通解 = 齐次通解 + 特解

例15.6：解\(y'' - 2y' + 5y = -e^x \cos2x\)

齐次方程：\(y'' - 2y' + 5y = 0\)

特征方程：\(r^2 - 2r + 5 = 0\)

根：\(r = 1 \pm 2i\)

齐次通解：\(y_h = e^x(C_1\cos2x + C_2\sin2x)\)

非齐次方程：\(y'' - 2y' + 5y = -e^x \cos2x\)

\(f(x) = -e^x \cos2x\)，\(\alpha = 1, \beta = 2\)

\(\alpha + \beta i = 1 + 2i\)是特征根（单根），所以\(k = 1\)

特解设为：\(y_p = x e^x (A\cos2x + B\sin2x)\)

代入方程求解\(A\)和\(B\)

计算得：\(A = 0, B = -\frac{1}{4}\)

特解：\(y_p = -\frac{x e^x \sin2x}{4}\)

通解：\(y = e^x(C_1\cos2x + C_2\sin2x) - \frac{x e^x \sin2x}{4}\)

4. 欧拉方程的求解

解题思路：

1. 确定是否为欧拉方程
2. 令\(x = e^t\)，\(t = \ln x\)
3. 将方程转化为常系数线性微分方程
4. 求解常系数线性微分方程
5. 代回\(x = e^t\)，得到原方程的通解

例15.7：解\(x^2y'' + 4xy' + 2y = 0 (x > 0)\)

令\(x = e^t\)，\(t = \ln x\)

\(\frac{dy}{dx} = \frac{1}{x} \frac{dy}{dt}\)

\(\frac{d^2y}{dx^2} = \frac{1}{x^2}(\frac{d^2y}{dt^2} - \frac{dy}{dt})\)

代入方程：

\(x^2\frac{1}{x^2}(\frac{d^2y}{dt^2} - \frac{dy}{dt}) + 4x\frac{1}{x} \frac{dy}{dt} + 2y = 0\)

\(\Rightarrow \frac{d^2y}{dt^2} - \frac{dy}{dt} + 4\frac{dy}{dt} + 2y = 0\)

\(\Rightarrow \frac{d^2y}{dt^2} + 3\frac{dy}{dt} + 2y = 0\)

特征方程：\(r^2 + 3r + 2 = 0\)

根：\(r = -1, r = -2\)

通解：\(y = C_1e^{-t} + C_2e^{-2t}\)

代回\(t = \ln x\)：\(y = \frac{C_1}{x} + \frac{C_2}{x^2}\)

四、易错点与注意事项

1. 伯努利方程的处理

注意\(n \neq 0, 1\)，否则不是伯努利方程
令\(z = y^{1-n}\)时，要正确求导

2. 非齐次方程特解的设定

确定\(\alpha\)是否为特征根
\(k = 0, 1, 2\)的正确应用
被积函数\(f(x)\)的类型要与特解形式对应

3. 欧拉方程的变换

注意\(x > 0\)的条件
变换时要正确求导：\(\frac{dy}{dx} = \frac{1}{x} \frac{dy}{dt}, \frac{d^2y}{dx^2} = \frac{1}{x^2}(\frac{d^2y}{dt^2} - \frac{dy}{dt})\)
代入后要化简方程

4. 特征根的判断

实根、重根、共轭复根的区分
不同情况下通解的形式不同

5. 待定系数法的计算

特解形式设定正确后，计算量较大
要仔细计算，避免符号错误

五、重要公式汇总

1. 一阶线性微分方程：
    通解：\(y = e^{-\int p(x)dx}[\int q(x)e^{\int p(x)dx}dx + C]\)
2. 伯努利方程：
    令\(z = y^{1-n}\)，转化为一阶线性微分方程
3. 二阶常系数齐次线性微分方程：
    通解：根据特征根的性质确定
4. 二阶常系数非齐次线性微分方程：
    通解 = 齐次方程通解 + 非齐次方程特解
    特解设法：\(y_p = x^k e^{\alpha x}Q_m(x)\)
5. 欧拉方程：
    令\(x = e^t\)，\(t = \ln x\)，转化为常系数线性微分方程

六、典型例题解析

例1：解一阶线性微分方程\(y' + 2xy = 2x\)

解题思路：

1. \(p(x) = 2x, q(x) = 2x\)
2. 积分因子\(\mu(x) = e^{\int2x dx} = e^{x^2}\)
3. 通解：\(y = e^{-x^2}[\int2x e^{x^2} dx + C]\)
4. 计算：\(\int2x e^{x^2} dx = e^{x^2} + C\)
5. 通解：\(y = e^{-x^2}(e^{x^2} + C) = 1 + Ce^{-x^2}\)

例2：解伯努利方程\(y' + y = y^2\)

解题思路：

1. \(n = 2\)，令\(z = y^{1-2} = y^{-1}\)
2. \(y = z^{-1}, y' = -z^{-2}z'\)
3. 代入方程：\(-z^{-2}z' + z^{-1} = z^{-2}\)
4. 乘以\(-z^2\)：\(z' - z = -1\)
5. 积分因子\(\mu(x) = e^{-x}\)
6. 通解：\(z = e^x[\int-e^{-x}dx + C] = e^x[e^{-x} + C] = 1 + Ce^x\)
7. 代回\(y = \frac{1}{z}\)：\(y = \frac{1}{1 + Ce^x}\)

例3：解二阶常系数非齐次线性微分方程\(y'' - 2y' + y = e^x\)

解题思路：

1. 齐次方程：\(y'' - 2y' + y = 0\)
特征方程：\(r^2 - 2r + 1 = 0\)
根：\(r = 1\)（重根）
齐次通解：\(y_h = (C_1 + C_2x)e^x\)
2. 非齐次方程：\(y'' - 2y' + y = e^x\)
\(f(x) = e^x\)，\(\alpha = 1\)，\(\alpha\)是特征方程的重根，所以\(k = 2\)
特解设为：\(y_p = Ax^2e^x\)
3. 代入求解\(A\)：
\(y_p' = A(2xe^x + x^2e^x)\)
\(y_p'' = A(2e^x + 4xe^x + x^2e^x)\)
代入方程：\(A(2e^x + 4xe^x + x^2e^x) - 2A(2xe^x + x^2e^x) + Ax^2e^x = e^x\)
化简：\(2Ae^x = e^x \Rightarrow A = \frac{1}{2}\)
4. 特解：\(y_p = \frac{1}{2}x^2e^x\)
5. 通解：\(y = (C_1 + C_2x)e^x + \frac{1}{2}x^2e^x\)
以上是第15讲"微分方程"的核心知识要点和做题思路，掌握这些内容可有效解决微分方程相关问题。
